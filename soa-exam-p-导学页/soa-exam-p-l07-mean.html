01
Ch7 · Lesson 7 · 课本 p.119

期望 / 均值 — 定义

期望是分布的"中心",是所有可能取值以概率为权重的加权平均,记作 \(E[X]=\mu\)。几何上它是密度曲线所围面积的质心(平衡点)——把概率想成质量,\(\mu\) 就是那根支起整片面积不倾倒的支点。

定义:概率加权平均 考频 ★★★

课本 p.119

把每个取值 \(x\) 乘上它的"权重"(概率 / 密度),再累加——期望就是分布的重心。

★ 必背Expected value — discrete / continuous $$E[X]=\sum_x x\,p(x)\qquad\qquad E[X]=\int_{-\infty}^{\infty} x\,f(x)\,dx$$
🗝️值乘权重加起来:概率是权,期望是重心 — 离散逐点加,连续换积分,同一思想。

离散对所有 \(p(x)\neq0\) 的 \(x\) 求和;连续对全轴积分。两式同一思想:权重之和为 1,故 \(E[X]\) 落在取值范围内。

连续:\(\mu\) 是密度面积的 质心(支点),右偏时 \(\mu\) 落在峰的右侧
离散:每根柱高 \(=p(x)\),\(\mu=\sum x\,p(x)\) 是柱群的平衡位置
例 7A课本 p.119 · 离散

\(X\) 为离散变量,\(p(x)=\dfrac{x}{15}\)(\(x=1,2,3,4,5\)),其余为 0。求 \(E[X]\)。

直接套定义:\(E[X]=\displaystyle\sum_{x=1}^{5} x\cdot\frac{x}{15}=\sum_{x=1}^{5}\frac{x^2}{15}=\frac{1+4+9+16+25}{15}=\frac{55}{15}=\frac{11}{3}\)。

E[X] = 11/3 ≈ 3.667
💡直觉:\(E[X]\) 不一定是某个"可能取到"的值(这里 \(11/3\) 就取不到)。它是长期重复试验结果的平均,是分布的重心而非某个观测。
📝对照概念:样本均值(sample mean)(课本 p.123 · Table 7.1 旁)——从 \(X\) 抽 \(n\) 个观测,加总除以 \(n\):\(\bar{X}=\dfrac{1}{n}\sum_{i=1}^{n}X_i\)。分清两兄弟:\(\mu=E[X]\) 是分布的理论均值(本章主角);\(\bar{X}\) 是一把样本的平均,本身是随机变量。计算器能直接出样本均值,但定义必须懂——\(\bar{X}\) 到 L22 中心极限定理时是主角(\(\bar{X}\approx N(\mu,\sigma^2/n)\))。
02
通用工具

LOTUS + 期望的线性

要算 \(g(X)\)(如 \(X^2\)、\(e^{X}\))的期望,不必先求出 \(g(X)\) 的分布——"无意识统计学家定律"让你直接对 \(g(x)\) 加权求和/积分。

LOTUS(Law of the Unconscious Statistician)考频 ★★★

课本 Lesson 7 · 求随机变量函数的期望

求 \(g(X)\) 的期望不必先求 \(g(X)\) 的分布:把 \(x\) 换成 \(g(x)\),权重 \(p(x)\)/\(f(x)\) 原地不动。

★ 必背E[g(X)] — 不必先求 g(X) 的分布 $$E[g(X)]=\sum_x g(x)\,p(x)\qquad\text{或}\qquad E[g(X)]=\int_{-\infty}^{\infty} g(x)\,f(x)\,dx$$
🗝️换头不换权 — 头(\(x\))换成 \(g(x)\),权(\(p\) / \(f\))一动不动。

方差、矩、矩母函数全靠它:\(E[X^2]=\int x^2 f(x)\,dx\)、\(E[e^{tX}]=\int e^{tx}f(x)\,dx\)。名字是玩笑:人人不假思索地当定义用,其实它是一条需要证明的定理。

期望的线性(Linearity)考频 ★★★

课本 Lesson 7 · 常用性质

常数可提出、加法可拆开——且 \(E[X+Y]=E[X]+E[Y]\) 对任意 \(X,Y\) 恒成立,不要求独立。

★ 必背Linearity of expectation $$E[aX+b]=a\,E[X]+b\qquad E[X+Y]=E[X]+E[Y]$$

独立性只在乘积 \(E[XY]\) 时才用得上。推论:线性组合 \(E\!\left[\sum a_i X_i\right]=\sum a_i E[X_i]\)——这是保险里"总赔付期望 = 各笔期望之和"的根据。

⚠️最大陷阱 — \(E[g(X)]\neq g(E[X])\):期望只对线性函数可以"穿进去"。非线性时不成立:一般 \(E[X^2]\neq (E[X])^2\)(两者之差正是方差 \(\ge 0\)),\(E\!\left[\tfrac1X\right]\neq\tfrac1{E[X]}\)。凸函数由 Jensen 不等式有 \(E[g(X)]\ge g(E[X])\)。
\(E[X^2]\) — 平方的平均\((E[X])^2\) — 平均的平方
算法LOTUS:\(\sum x^2\,p(x)\) 或 \(\int x^2 f(x)\,dx\)先算 \(E[X]\),再整体平方
一句话先平方、后加权平均先加权平均、后平方
大小关系恒有 \(E[X^2]\ge (E[X])^2\),两者之差就是方差 \(\operatorname{Var}(X)\)(下一章主角)

Ch7 · 高频考点

生存函数求期望(捷径)

对非负随机变量,期望还能对生存函数 \(S(x)=1-F(x)\) 积分/求和得到。题面给 \(F\) 时它最香:\(F\to S\) 只是减法,免去「先求导得 \(f\)、再乘 \(x\) 积分」的两步绕路——考试极爱考。为什么成立?见 §06 直觉拓展的积木图。

生存函数公式(Survival-function formula)考频 ★★★

课本 p.120 · 式 (7.1) · 分部积分证得

非负变量的期望 = 生存函数下的面积:对 \(1-F\) 积分,不必先求密度再乘 \(x\)。

★ 必背E[X] via survival function — continuous $$E[X]=\int_0^\infty \big(1-F(x)\big)\,dx=\int_0^\infty S(x)\,dx$$

离散非负整数值变量的对应式:

E[X] via survival — discrete (nonneg integer) $$E[X]=\sum_{k\ge 0} P[X>k]=\sum_{k\ge 0}\big(1-F(k)\big)$$
🗝️按层切蛋糕 — 把期望横着切:每层 \(dx\) 的贡献是"还活着"的概率 \(S(x)\),层层叠起来就是 \(E[X]\)。注意 \(S\) 不是被切的曲线,是每层的读数——被切的是哪块面积,见 §06。

前提:变量非负(连续要求 \(f(x)=0\) 当 \(x<0\))。题目给分段常数 \(F\) 时,只需对 \(1-F\) 分段"数长度"求和,免去先微分求密度再乘 \(x\) 积分。

展开 · 分部积分的证明梗概

对 \(\int_0^\infty x f(x)\,dx\) 分部积分,令 \(u=x,\ dv=f(x)\,dx\),边界项在 \(\lim_{x\to\infty}x\big(1-F(x)\big)=0\)(均值有限时)时消失,余下 \(\int_0^\infty(1-F(x))\,dx\)。

例 7D课本 p.121 · 离散生存函数

离散 \(N\):\(f(0)=0.2,\ f(1)=0.3,\ f(2)=0.4,\ f(3)=0.1\)。用生存函数法求 \(E[N]\)。

先求 \(P[N>k]\):\(P[N>0]=1-0.2=0.8\);\(P[N>1]=1-0.5=0.5\);\(P[N>2]=1-0.9=0.1\);\(P[N>3]=0\)。
故 \(E[N]=0.8+0.5+0.1=1.4\)。与直接法 \(0(0.2)+1(0.3)+2(0.4)+3(0.1)=1.4\) 一致。

E[N] = 1.4
💡为什么好用:题目常直接给 \(F(x)\)(如分段常数),生存函数法只需对 \(1-F\) 分段"数长度"求和,免去先微分求密度再乘 \(x\) 积分。保险里的 \(E[\min(X,k)]=\int_0^k(1-F(x))\,dx\)(式 7.2,免赔额/限额)正是它的直接推论。
例 7E课本 p.122 · 式 (7.2) 限额赔付

保单损失 \(X\) 的分布函数 \(F(x)=0\)(\(x<0\)),\(=\sin x\)(\(0\le x<\tfrac{\pi}{2}\)),\(=1\)(\(x\ge\tfrac{\pi}{2}\));给付限额 0.5。求每次损失的期望赔付 \(E[\min(X,0.5)]\)。

直接套式 (7.2),只积到限额 \(k=0.5\) 就停:
\(\displaystyle E[\min(X,0.5)]=\int_0^{0.5}\big(1-\sin x\big)\,dx=\Big[x+\cos x\Big]_0^{0.5}=0.5+\cos 0.5-1\approx 0.3776\)。

E[min(X,0.5)] ≈ 0.377583
📝对比定义法:不用式 (7.2) 就得拆两截——\(\int_0^{0.5}x f(x)\,dx + 0.5\,S(0.5)\)(先微分求 \(f=\cos x\) 再分部积分)。生存函数法一步到位,这正是它"考试极爱考"的原因。

Ch7 · 课本 p.122

混合型随机变量求期望

质量点 + 连续密度混搭的变量(L05 §5.1 说过:含免赔额的赔付天然是这种),期望分两截算再相加——离散部分乘加,连续部分积分。

混合型期望公式 考频 ★★

课本 p.122 · 例 7F 前理论段

哪部分是质量点就按离散办,哪部分有密度就按连续办,最后相加。

★ 必背E[X] — mixed random variable $$E[X]=\sum_i x_i\,P[X=x_i]\;+\;\int x\,f(x)\,dx$$

自检:质量点概率之和 + 密度积分必须 \(=1\),动手前先验一遍,防止漏读题面某一截。混合型的 \(F\) 长相:跳点(质量)+ 斜坡(密度),对照 L05 §04 的阶梯/S 型直觉。

例 7F课本 p.122 · 混合型精解

\(X=5\) 的概率为 0.25;其余取值按密度 \(f(x)=1.5x\)(\(0\le x\le 1\),其他为 0)。求 \(E[X]\)。

① 验总概率:\(0.25+\displaystyle\int_0^1 1.5x\,dx=0.25+0.75=1\) ✓。
② 质量点截:\(5\times 0.25=1.25\)。③ 连续截:\(\displaystyle\int_0^1 x\cdot 1.5x\,dx=\frac{1.5x^3}{3}\Big|_0^1=0.5\)。
相加:\(E[X]=1.25+0.5=1.75\)。

E[X] = 1.75
💡保费术语(课本同段):保险公司一年总赔付(aggregate losses)的期望叫净保费(net premium);在此之上加费用与利润,实际收的叫毛保费(gross premium)。读题看到 net premium,直接翻译成"求期望赔付"。
05
边界情形

期望不一定存在

定义里的和 / 积分可能发散——此时期望(以及更高阶的矩)不存在。别默认任何分布都有有限均值。

存在性(Existence)考频 ★

课本 p.123 · Table 7.1 旁注

和 / 积分不收敛,该阶矩就不存在——尾巴太肥的分布先验证收敛再谈期望。

"若计算某阶矩的和或积分不收敛,该矩就不存在。"典型如柯西分布 \(f(x)=\dfrac{1}{\pi(1+x^2)}\):\(\int|x|f(x)\,dx=\infty\),均值不存在;又如 \(f(x)\propto x^{-2}\) 的重尾损失,均值可能发散。

实操:遇到 \(x\to\infty\) 处密度衰减太慢(尾巴太肥)时,先验证收敛再谈期望值。生存函数公式里的边界项 \(x(1-F(x))\to0\) 同样要求均值有限才成立。

06
直觉 · 拓展

换个角度看懂它

💡 直觉例子一:天平上的砝码(期望 = 重心)

在一根无重量的直尺上,把质量为 \(p(x)\) 的砝码放在刻度 \(x\) 处(总质量为 1)。用一根手指把尺子顶住不翻的那个支点,恰好在 \(\mu=\sum x\,p(x)\)——因为支点处力矩平衡:\(\sum (x-\mu)\,p(x)=0\) 恰好等价于 \(\mu=E[X]\)。

重心:例 7D 的 pmf 当砝码,支点顶在 \(\mu=1.4\) 恰好平衡
右偏:尾巴那点质量力臂长,把支点(均值)拽到中位数右边

连续版把砝码换成"质量密度 \(f(x)\)",支点是密度面积的质心。这也解释了"右偏分布均值被长尾拉向右":远处哪怕一点点质量,力臂很长,也能把支点拽过去。

💡 直觉例子二:按层切蛋糕(生存函数为什么给期望)

第一步:先画对被切的东西。给每个结果画一根柱子——宽 = 概率,高 = 取值(柱子按高矮排好队,横切才连成整层;排序不改变总面积)。这堆积木的总面积 = \(\sum(\text{高}\times\text{宽}) = \sum_k k\,p(k) = E[N]\)。注意:它不是 \(f\) 的图像,也不是 \(F\) 的图像——是"取值×概率"自己攒出来的面积。没画对这块区域,两种切法就对不上。

第二步:同一块面积,数两遍。竖着数按柱:每根柱子面积 = 取值×概率。横着切按层:第 \(k\) 层是条高 1 的横带,长度 = "够到这层"的柱子总宽 = \(P[N\ge k]\)——恰好是生存概率。层层相加得 \(E[N]=\sum_{k\ge0}P[N>k]\),与竖着数必然相等。

竖着数:\(\sum k\,p(k)=0.3+0.8+0.3=1.4\)(\(N{=}0\) 那根高 0,零贡献)
横着切:\(\sum P[N>k]=0.8+0.5+0.1=1.4\)

左右是同一块阶梯形面积(例 7D 的分布,取值大的排左边):竖切按"取值×概率"数,横切按"每层生存概率"数,殊途同归 \(=E[N]=1.4\)。看清 \(S\) 的角色——它不是被切的曲线,而是横切每层的读数。连续版每层厚 \(dx\)、宽 \(S(x)\),叠起来就是 \(E[X]=\int_0^\infty S(x)\,dx\)。人均寿命口径同理:按人算 = 每人寿命×人口占比(竖切);按年算 = 每年存活率相加(横切)——"生存函数"因此得名。

💡 直觉例子三:一条链看清"求导 vs 积分"

求 \(E[X]\) 一会求导、一会积分,不是两套方法,是同一条链上相反方向的两步:\(F\) 是概率存量(攒到 \(x\) 为止攒了多少),\(f\) 是流量(攒的速度),\(E[X]\) 是用 \(f\) 加权的汇总。生存捷径好用的根源一句话:\(F\to S\) 只是减法,\(F\to f\) 才要求导。

一条链:求导、积分是链上相反方向的两步;捷径绕开求导那一步
题给什么怎么走步数
\(f(x)\)直接 \(\int x\,f(x)\,dx\)1
\(F(x)\),\(X\) 非负捷径 \(\int_0^\infty(1-F)\,dx\)1(免求导)
\(F(x)\),走常规先 \(F'=f\),再 \(\int x\,f\,dx\)2(互逆两步,绕路)
分段常数 \(F\)对 \(1-F\) 分段"数长度"相加1

一会分不清三兄弟就回 L05:\(f\) 读面积(要积分才是概率,密度值可 >1),\(F/S\) 读高度(曲线高度本身就是概率)。口诀:f 管密度、F 管以下、S 管以上;\(F+S=1\),\(f\) 是坡度(\(F'=f,\ S'=-f\))。另:本链求导的对象是 \(x\);L32 的 MGF 取矩是对 \(t\) 求导——两条链别混。

🎯 SOA 题型套路

  1. 保险高频三件套:免赔额(deductible)/ 给付限额(policy limit)/ 期望赔付(expected payment)——见到 "payment per loss, up to a maximum of k" 先写 \(E[\min(X,k)]=\int_0^k S(x)\,dx\)。
  2. 题面给 \(F\)(尤其分段常数)不给 \(f\) → 直接走生存函数法,别费劲微分。
  3. 给 \(E[X]\) 求 \(E[aX+b]\) 是送分的线性;一旦函数非线性(\(X^2\)、\(1/X\)、\(e^X\))必须 LOTUS 老老实实积,严禁把期望塞进函数里。
  4. 反解型:给 \(E[X]\) 的值和带参数的 \(p(x)\),回头解参数(练 7.1)。

易错点 + 自测

⚠️ 高频易错点

  • \(E[g(X)]\neq g(E[X])\):非线性函数不能把期望"套进去"(Jensen),\(E[X^2]\neq(E[X])^2\)。
  • 生存函数上下限搞混:整段期望积到 \(\infty\),\(E[\min(X,k)]\) 只积到 \(k\);且公式要求变量非负。
  • 忘了均值可能不存在:重尾 / 发散积分,先验收敛。
  • \(E[X+Y]\) 误加独立条件:线性对任意变量成立,不需独立。

收尾

公式速查卡 · 均值

学完本章,这张卡随手能默写,就算过关了。

定义 / LOTUS / 线性

\(E[X]=\sum_x x\,p(x)=\int x f(x)\,dx\)

\(E[g(X)]=\sum g(x)p(x)=\int g(x)f(x)\,dx\)

\(E[aX+b]=aE[X]+b\)

\(E[X+Y]=E[X]+E[Y]\)(不需独立)

生存函数 / 边界

\(E[X]=\int_0^\infty(1-F(x))\,dx\)(非负)

\(E[X]=\sum_{k\ge0}P[X>k]\)(离散)

\(E[\min(X,k)]=\int_0^k(1-F(x))\,dx\)

混合型:\(E[X]=\sum_i x_i P[X{=}x_i]+\int x f(x)\,dx\)

\(E[g(X)]\neq g(E[X])\);积分发散则均值不存在

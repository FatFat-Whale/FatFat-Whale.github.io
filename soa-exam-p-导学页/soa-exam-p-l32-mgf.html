第三十二章 · Moment Generating Function 矩母函数

给每个随机变量配一把"万能钥匙" \(M_X(t)=E[e^{tX}]\)。它在 0 处求导就吐出各阶矩(均值、方差全在里头)、独立随机变量相加就变成 MGF 相乘、而且一个 MGF 唯一确定一个分布——认出它就等于认出分布。对照课本 Lesson 32 精读。

★ 本章拿走 3 句话

  1. MGF 是分布的指纹:\(M_X(t)=E[e^{tX}]\),一个 MGF 唯一对应一个分布——认出形状就认出分布。

01
Ch32 · Lesson 32 · 课本 p.505

矩母函数 — 定义

给随机变量 \(X\) 配一个以 \(t\) 为自变量的函数:把 \(X\) 塞进 \(e^{tX}\) 再取期望。它在 \(t=0\) 处一定有定义(\(M_X(0)=E[e^0]=1\)),但对别的 \(t\) 未必收敛。

定义:矩母函数 \(M_X(t)\) 考频 ★★★

课本 p.505 · 式 (32.1)

把 \(X\) 塞进 \(e^{tX}\) 再取期望,得到一个 \(t\) 的函数——分布的全部信息都被压进这一个函数里。

★ 必背Moment Generating Function $$M_X(t)=E\!\left[e^{tX}\right]$$
🗝️M(0)=1 恒成立 — 任何 MGF 代 \(t=0\) 必得 1,算完先拿这条自检。

对离散变量按概率加权:\(M_X(t)=\sum_x e^{tx}\,p(x)\);对连续变量积分:\(M_X(t)=\int_{-\infty}^{\infty} e^{tx} f(x)\,dx\)。

"看着吓人,做过一个例子就不怕了。"注意它是 \(t\) 的函数,不是 \(X\) 的函数——\(X\) 已被期望积掉。

Example 32A课本 p.505 · 离散

离散 \(X\) 取值 \(0,1,2\),概率分别 \(0.5,0.3,0.2\)。写出 \(M_X(t)\) 并求 \(M(3)\)。

直接按定义加权:\(M(t)=E[e^{tX}]=0.5e^{0}+0.3e^{t}+0.2e^{2t}=0.5+0.3e^{t}+0.2e^{2t}\)。
代 \(t=3\):\(M(3)=0.5+0.3e^{3}+0.2e^{6}=0.5+6.0257+80.6857=87.21142\)。

M(t)=0.5+0.3eᵗ+0.2e²ᵗ　M(3)=87.21142
Example 32B课本 p.505 · 连续(指数)

\(X\) 服从均值 \(\theta\) 的指数分布,\(f(x)=\tfrac1\theta e^{-x/\theta}\ (x\ge0)\)。求 \(M_X(t)\)。

按定义积分:\(M(t)=\displaystyle\int_0^\infty e^{tx}\,\frac{e^{-x/\theta}}{\theta}\,dx=\int_0^\infty \frac{e^{(t-1/\theta)x}}{\theta}\,dx\)。仅当 \(t-\tfrac1\theta<0\)(即 \(t<\tfrac1\theta\))积分收敛,得 \(M(t)=\dfrac{1}{1-\theta t}\)。

M(t)=1/(1−θt),　t < 1/θ
💡定义域会变小:指数 MGF 只在 \(t<1/\theta\) 有定义——考试给闭式 MGF 时常附一句 "\(-\infty<t<1/\theta\)",这个收敛区间本身就是提示分布的线索。
02
核心用途一

取矩:在 0 处求导

名字里的"矩"(moment)就是这么来的——MGF 在 \(t=0\) 处求 \(n\) 次导,恰好剥出第 \(n\) 阶原点矩 \(E[X^n]\)。

MGF 生成矩(generates moments)考频 ★★★

课本 p.506 · 式 (32.2)

对 \(t\) 求 \(n\) 次导、再令 \(t=0\),吐出第 \(n\) 阶原点矩。

★ 必背n-th raw moment $$M_X^{(n)}(0)=E\!\left[X^{n}\right]\qquad\Longrightarrow\qquad E[X]=M'_X(0),\ \ E[X^2]=M''_X(0)$$
🗝️先求导、后代零 — 导完还是 \(t\) 的函数,代 \(t=0\) 才是矩。

关键:一定在 \(t=0\) 处求值。先对 \(t\) 求导得到一个 \(t\) 的函数,再代 \(t=0\)。方差随之而来:\(\mathrm{Var}(X)=M''(0)-\big(M'(0)\big)^2\)。

展开 · 为什么 0 处求导剥出矩

把 \(e^{tX}\) 展开成 \(1+tX+\tfrac{t^2}{2!}X^2+\tfrac{t^3}{3!}X^3+\cdots\),取期望得 \(M(t)=1+tE[X]+\tfrac{t^2}{2!}E[X^2]+\cdots\)。对 \(t\) 求 \(n\) 次导后,\(E[X^n]\) 那一项恰好变成常数,其余项都还带着 \(t\)——代 \(t=0\) 全部消失,只剩 \(E[X^n]\)。

MGF 是分布 ↔ 矩之间的枢纽
曲线过 \((0,1)\);\(t=0\) 处切线斜率 = 均值
Example 32C课本 p.506 · 求方差

已知 \(M_X(t)=\dfrac{1}{(1-5t)^{3}}=(1-5t)^{-3}\)。求 \(\mathrm{Var}(X)\)。

求一阶导:\(M'(t)=\dfrac{(3)(5)}{(1-5t)^{4}}=\dfrac{15}{(1-5t)^4}\),故 \(M'(0)=15=E[X]\)。
求二阶导:\(M''(t)=\dfrac{(5^2)(3)(4)}{(1-5t)^{5}}=\dfrac{300}{(1-5t)^5}\),故 \(M''(0)=300=E[X^2]\)。
于是 \(\mathrm{Var}(X)=E[X^2]-(E[X])^2=300-15^{2}=300-225=75\)。

E[X]=15　E[X²]=300　Var(X)=75
⚠️最常翻车处:忘了代 \(t=0\)。\(M'(t)\) 还是 \(t\) 的函数,不是 \(E[X]\);必须先求导、后令 \(t=0\)。也别把 \(M''(0)\) 当成方差——那是 \(E[X^2]\),还要减 \((E[X])^2\)。
03
核心用途二 + 唯一性

独立和 = MGF 相乘

MGF 是乘性的:独立随机变量相加,MGF 就相乘。配合"一个 MGF 唯一确定一个分布",相加后认出乘积的形状,就直接读出和的分布——省去卷积积分。

独立和的 MGF(multiplicative)考频 ★★★

课本 p.506 · 式 (32.3)

独立随机变量相加,MGF 相乘——"加法变乘法",免掉卷积。

★ 必背Sum of independent r.v.'s $$X\perp Y\ \Longrightarrow\ M_{X+Y}(t)=M_X(t)\,M_Y(t)$$

独立是前提——不独立时期望不能拆成乘积。

展开 · 一行推导

\(M_{X+Y}(t)=E[e^{t(X+Y)}]=E[e^{tX}e^{tY}]\overset{\text{独立}}{=}E[e^{tX}]\,E[e^{tY}]=M_X(t)M_Y(t)\)。

唯一性(uniqueness):MGF 决定分布 考频 ★★

课本 p.506

MGF 是"认分布的指纹":两个变量的 MGF 处处相等 ⟺ 它们同分布。

每个分布有唯一的 MGF;反过来,若两个变量的 MGF 处处相等,则它们同分布。把和的 MGF 化简后,对照常见分布表认出它是谁。

常见闭合结果:独立同参指数之和 → 伽马;独立正态之和 → 正态(\(\mu\) 相加、\(\sigma^2\) 相加);独立泊松之和 → 泊松(\(\lambda\) 相加);独立二项(同 \(p\))之和 → 二项。

相加变相乘——独立是前提
唯一性:MGF 相等 ⟺ 分布相同
Example 32D课本 p.507 · 独立指数之和

\(X\) 指数、均值 2;\(Y\) 指数、均值 5;\(X,Y\) 独立。求 \(X+Y\) 的 MGF。

由 Example 32B,\(M_X(t)=\dfrac{1}{1-2t}\),\(M_Y(t)=\dfrac{1}{1-5t}\)。独立相乘:

$$M_{X+Y}(t)=M_X(t)M_Y(t)=\frac{1}{(1-2t)(1-5t)}$$

M_{X+Y}(t) = 1 / [(1−2t)(1−5t)]

联合矩母函数(joint MGF)

课本 p.507

MGF 推广到两个变量:一个函数装下 \((X,Y)\) 的全部混合矩。

★ 必背Joint MGF $$M_{X,Y}(t_1,t_2)=E\big[e^{t_1X+t_2Y}\big]$$

取矩:对 \(t_1\) 偏导 \(k\) 次、对 \(t_2\) 偏导 \(m\) 次,再在 \((0,0)\) 取值,得 \(E[X^kY^m]\)。最常用的是混合偏导:

$$\frac{\partial^2 M}{\partial t_1\,\partial t_2}\bigg|_{(0,0)}=E[XY]$$

这是算协方差的入口:\(\mathrm{Cov}(X,Y)=E[XY]-E[X]E[Y]\),三个期望都能从同一个 \(M_{X,Y}\) 求偏导取出。另外令 \(t_2=0\) 即退化回边缘 MGF:\(M_{X,Y}(t_1,0)=E[e^{t_1X}]=M_X(t_1)\)。

常见分布的 MGF

考试给一个闭式 MGF,考的往往是"你认不认得出它是谁"。下表按唯一性把常见分布的 MGF 一网打尽——认出形状 = 认出分布。指数/伽马这里按均值 \(\theta\) 参数化(课本口径);若按速率 \(\lambda=1/\theta\) 参数化,\(\tfrac{1}{1-\theta t}=\tfrac{\lambda}{\lambda-t}\),两写法等价。

分布MGF \(M_X(t)\)定义域均值 / 方差
二项 Binomial\((n,p)\)\((1-p+pe^{t})^{n}\)所有 \(t\)\(np\) / \(np(1-p)\)
泊松 Poisson\((\lambda)\)\(e^{\lambda(e^{t}-1)}\)所有 \(t\)\(\lambda\) / \(\lambda\)
几何 Geometric\((p)\)\(\dfrac{pe^{t}}{1-(1-p)e^{t}}\)\(e^{t}<\tfrac{1}{1-p}\)\(\tfrac1p\) / \(\tfrac{1-p}{p^2}\)

🗝️求导取矩、相乘拼和、认脸识分布 — MGF 三大用法,一句默完。
🔗用唯一性做加法:看到 \(\dfrac{1}{(1-\theta t)^{n}}\) 就知道是伽马——恰是 \(n\) 个均值 \(\theta\) 的独立指数之和(指数 MGF 的 \(n\) 次方)。看到 \(e^{\mu t+\sigma^2t^2/2}\) 就是正态,独立正态相加时 \(\mu\)、\(\sigma^2\) 各自相加。
📝连向中心极限定理:课本指出——独立随机变量之和的 MGF 是乘积,取极限后逼近正态的 MGF \(e^{\mu t+\sigma^2t^2/2}\),这正是 CLT(中心极限定理)证明背后的思路。MGF 是连接"求和"与"正态"的桥。
05
直觉 · 拓展

换个角度看懂它

💡 直觉例子:MGF 是分布的"条形码"

超市里每件商品有一条唯一的条形码;每个分布也有唯一的 \(M(t)\)。收银台不看商品长相,扫码即识别——考试不用做卷积,把 MGF 化简成认识的形状就识别了分布。比如看到 \((1-5t)^{-3}\):套 \((1-\theta t)^{-\alpha}\) 立刻读出伽马 \(\alpha=3,\ \theta=5\),连均值 \(\alpha\theta=15\) 都不用求导(与 Example 32C 的 \(M'(0)=15\) 吻合)。

扫码认形:例 32C 的 \((1-5t)^{-3}\) 逐位对上模板,直接读出 \(\alpha=3,\ \theta=5\)
换域好办事:难算的卷积/积分切到 \(t\) 域变乘法/求导,唯一性保证零失真

更深一层:\(M(t)=E[e^{tX}]\) 本质是给密度做了一次"变换域"处理——从"\(x\) 的世界"切到"\(t\) 的世界"。信息一点不丢(唯一性),但难算的操作全变简单:求矩变成求导、独立卷积变成乘法。这是数学里"换个域好办事"的经典手法。

🎯 SOA 题型套路

  1. 给 MGF 认分布:化简 → 对照 Table 32.1 认形状,收敛区间(如 \(t<1/\theta\))也是线索。
  2. 求矩:先求导、后代 \(t=0\);若形状恰是表中分布,直接抄均值/方差更快(如 \((1-2500t)^{-4}\) 是伽马,\(E[X]=\alpha\theta\))。
  3. 独立和:MGF 相乘后再认形——泊松×泊松 → \(\lambda\) 相加,正态×正态 → \(\mu,\sigma^2\) 各自相加。
  4. 陷阱:\(M''(0)\) 不是方差、导完忘代 \(t=0\)、不独立乱相乘;\(M(0)=1\) 随时用来自检。

巩固

易错点 + 自测

⚠️ 高频易错点

  • 求矩忘代 \(t=0\):先求导得 \(t\) 的函数,再令 \(t=0\) 才是矩。
  • \(M''(0)\) 不是方差:它是 \(E[X^2]\),还要减 \((E[X])^2\)。
  • 用乘积必须独立:\(M_{X+Y}=M_XM_Y\) 只在 \(X\perp Y\) 成立。
  • 常数项含 \(t=0\) 归一:离散 MGF 各系数=概率,和为 1;\(M(0)=1\) 恒成立,可自查。
  • 定义域别丢:指数/伽马 MGF 只在 \(t<1/\theta\) 有定义。

一台双部件设备:连续联合分布全景 L24–L31

连续联合分布这一区块,公式全都不难——难的从来是那张图。所有题共用一个动作:画区域 → 定内外限 →(必要时)换序。剩下三个词就讲完了:边缘=把概率的帐篷往一面墙上压扁;条件=在帐篷上切一刀,再把切片归一成新的密度;矩=在帐篷上给每点挂个权重 $g$ 再积。本页用一台"主件 + 备件"的设备把 L24–L31 八课串成一条线。

💡 概念标签可点弹卡;所有数字手算可验;E02 管"认分布",本页管"两个变量同时上场时,先画图再动笔"。

配套章节: L24 联合密度 · L25 联合均匀 · L26 边缘 · L27 联合矩 · L28 协方差 · L29 条件分布 · L30 条件矩 · L31 二元正态

第一幕 · 全景

一张地图,五个动作

全景一个动作打天下:画区域 → 定限 →(必要时)换序

唯一的底层事实:概率是密度曲面下的体积——

$$P\bigl[(X,Y)\in D\bigr]=\iint_{D\,\cap\,\text{支撑}} f(x,y)\,dA$$

此后每个动作都是这张图上的一个几何操作:

动作几何含义 → 公式
求概率 $P[(X,Y)\in D]$阴影 = $D\cap$支撑,二重积分;联合均匀时退化为面积比(免积分)
求边缘 $f_X(x)$压扁:把帐篷压向 x 墙,$f_X(x)=\int f(x,y)\,dy$,上下限从图上读
求条件 $f_{Y\mid X}(y\mid x)$切片归一:在 $x$ 处切一刀,$f_{Y\mid X}=f(x,y)/f_X(x)$
求矩 $E[g(X,Y)]$每点挂权重再积:$\iint g\cdot f\,dA$(LOTUS 二维版)
判独立密度可分离 × 支撑是矩形,两条缺一不可
命门在图不在式:五个动作的公式加起来不到三行;真正决定对错的是那张支撑图——内外限、切片长度、压扁方向,全从图上读。
第二幕 · 分组对比

四张对照表:纪律、判据、两种特权

对比①定限三步纪律 + 换序示范

三步纪律:① 把每条不等式画成边界线,涂出支撑;② 选切法定限——内限可以含另一变量,外限必须是纯数;③ 要换序,回到图上重读限,不是机械对调上下限。

示范:同一片三角($f=2$ 于 $0<x<y<1$),验证总概率 $=1$,两种次序并排写:

切法写法内层积完结果
竖切(先积 $y$)$\displaystyle\int_0^1\!\!\int_x^1 2\,dy\,dx$$\displaystyle\int_0^1 2(1-x)\,dx$$1$ ✓
横切(先积 $x$)$\displaystyle\int_0^1\!\!\int_0^y 2\,dx\,dy$$\displaystyle\int_0^1 2y\,dy$$1$ ✓
对照看内限:竖切内限 $x\to 1$、横切内限 $0\to y$——都直接抄自链式不等式 $0<x<y<1$ 的两半。内限带变量、外限是纯数,写反必错。
对比②独立判定:两条缺一不可

独立 $\iff f(x,y)=f_X(x)\,f_Y(y)$,实操拆成两问:密度可分离成 $g(x)h(y)$ 吗?支撑是矩形吗?

密度可分离?支撑矩形?独立?
$4xy$ 于 $(0,1)^2$✓ $=4x\cdot y$✓独立
$x+y$ 于 $(0,1)^2$✗ 和拆不开✓不独立
$2$ 于 $0<x<y<1$✓ 常数✗ 三角不独立(本页主角)
三角支撑必不独立:知道 $X=0.9$ 就把 $Y$ 压进了 $(0.9,1)$——一个变量的取值改变另一个的活动范围,谈何独立。密度再漂亮也救不了歪支撑。
对比③联合均匀特权:概率 = 面积比

密度是常数 $f\equiv 1/\text{area}(S)$ 时,一切积分退化为量面积:

要什么特权算法
概率$P=\dfrac{\text{area}(D\cap S)}{\text{area}(S)}$——免积分,尺子量出来
密度高度$1/\text{area}(S)$(体积必须为 1)
边缘未必均匀:边缘 $\propto$ 切片长度——三角支撑压扁出线性边缘
条件切片上仍均匀:常数 ÷ 常数还是常数
特权的边界:面积比只对"密度是常数"成立;题面一句 "uniformly distributed on region…" 就是发特权卡,不用白不用——但边缘均匀这条它不送。
对比④二元正态特权:直线、恒方差、零相关⇔独立
特权内容
条件期望是直线$E[Y\mid X{=}x]=\mu_Y+\rho\dfrac{\sigma_Y}{\sigma_X}(x-\mu_X)$——回归直线,免积分
条件方差恒定$\operatorname{Var}(Y\mid X{=}x)=\sigma_Y^2(1-\rho^2)$,与 $x$ 无关
条件分布仍正态切片还是钟形,可继续查表算概率
零相关 ⇔ 独立仅正态成立;一般分布只有"独立 ⇒ 零相关"单向
特权别乱借:"$\rho=0$ 所以独立"只在二元正态世界合法;一般分布里零相关照样可以强依赖(如 $Y=X^2$)。
第三幕 · 贯穿场景

底座:一台双部件设备,f(x,y) = 2 于 0 < x < y < 1

某设备装主件(寿命 $X$,千小时)与备件($Y$):备件规格更高,设计上必然比主件后坏,故支撑是三角 $0<x<y<1$,联合密度取常数 $f(x,y)=2$(三角面积 $\tfrac12$,体积正好归一)——这正是三角上的联合均匀,第二幕的两张特权表当场开用。

Q1画区域求 P[Y > 2X]:积分与面积比双算

质保部想知道"备件寿命超过主件两倍"的概率:求 $P[Y>2X]$。

先画图,再用两种次序各积一遍,最后面积比对账。

  1. 画区域:$y>2x$ 在三角内自动蕴含 $y>x$,所求区域顶点 $(0,0),(0,1),(\tfrac12,1)$。
  2. 竖切($x$ 只能到 $\tfrac12$,内限 $2x\to1$):$\displaystyle\int_0^{1/2}\!\!\int_{2x}^{1}2\,dy\,dx=\int_0^{1/2}2(1-2x)\,dx=\Bigl[2x-2x^2\Bigr]_0^{1/2}=1-\tfrac12=\mathbf{\tfrac12}$。
  3. 横切换序重读限(内限 $0\to y/2$,因为 $x<y/2$ 比 $x<y$ 更紧):$\displaystyle\int_0^{1}\!\!\int_{0}^{y/2}2\,dx\,dy=\int_0^1 y\,dy=\tfrac12$ ✓。
  4. 面积比对账(联合均匀特权):$P=\dfrac{1/4}{1/2}=\tfrac12$ ✓——三种算法一个答案。
换序不是对调:竖切外限是 $0\to\tfrac12$,横切外限却是 $0\to1$——上下限全变了。换序 = 回到图上换个方向重新切,别拿旧限硬搬。
Q2边缘密度:把帐篷压向两面墙

采购部只关心主件:"$X$ 单独看服从什么分布?"仓库只关心备件 $Y$。

求 $f_X(x)$ 与 $f_Y(y)$,积分限从图上读。

  1. 压向 x 墙(对 $y$ 积,竖切片长度从图读:$x\to1$):$f_X(x)=\displaystyle\int_x^1 2\,dy=\mathbf{2(1-x)},\ 0<x<1$。
  2. 压向 y 墙(对 $x$ 积,横切片 $0\to y$):$f_Y(y)=\displaystyle\int_0^y 2\,dx=\mathbf{2y},\ 0<y<1$。
  3. 体检:$\int_0^1 2(1-x)dx=1$ ✓,$\int_0^1 2y\,dy=1$ ✓;联合均匀,边缘却都不均匀——切片长度在变。
边缘=切片长度 × 密度高度:常数密度下边缘形状就是"切片长度函数"。主件偏短寿($f_X$ 递减)、备件偏长寿($f_Y$ 递增)——全是三角支撑逼出来的。
Q3条件密度与条件期望:切一刀,归一,取重心

运维实测:主件在 $400$ 小时($x=0.4$)时坏了。备件还能撑多久?

求 $f_{Y\mid X}(y\mid 0.4)$ 与 $E[Y\mid X=0.4]$。

  1. 切片归一:$f_{Y\mid X}(y\mid 0.4)=\dfrac{f(0.4,y)}{f_X(0.4)}=\dfrac{2}{2(1-0.4)}=\mathbf{\dfrac{5}{3}},\quad 0.4<y<1$。
  2. 认形状:常数高度 → $Y\mid X{=}0.4\sim U(0.4,\,1)$——联合均匀切片仍均匀(第二幕特权③)。
  3. 条件期望=均匀中点:$E[Y\mid X{=}0.4]=\dfrac{0.4+1}{2}=\mathbf{0.7}$(备件预计撑到 700 小时);顺手 $\operatorname{Var}=\dfrac{(1-0.4)^2}{12}=0.03$。
条件密度的分母永远是"已知那侧"的边缘:切在 $x$ 上就除 $f_X(x)$——这正是连续版贝叶斯的结构:联合 ÷ 边缘 = 条件。切片支撑从图上读,别写成 $0<y<1$。
Q4协方差:三角支撑逼出正相关

可靠性工程师问:主件长寿的设备,备件是不是也更长寿?求 $\operatorname{Cov}(X,Y)$,先猜符号再算。

用 $E[XY]-E[X]E[Y]$ 算,并说清正负的直觉。

  1. 先猜:支撑是 $y>x$——$X$ 大时 $Y$ 被抬到更高的地板上,应为正相关。
  2. 两个边缘均值(用 Q2 结果):$E[X]=\displaystyle\int_0^1 x\cdot2(1-x)\,dx=2\bigl(\tfrac12-\tfrac13\bigr)=\tfrac13$;$E[Y]=\displaystyle\int_0^1 y\cdot 2y\,dy=\tfrac23$。
  3. 混合矩(竖切):$E[XY]=\displaystyle\int_0^1\!\!\int_x^1 2xy\,dy\,dx=\int_0^1 x(1-x^2)\,dx=\tfrac12-\tfrac14=\tfrac14$。
  4. 合成:$\operatorname{Cov}=\tfrac14-\tfrac13\cdot\tfrac23=\tfrac{9-8}{36}=\mathbf{\tfrac{1}{36}}>0$ ✓ 与直觉一致。
  5. 加餐:$\operatorname{Var}(X)=\tfrac16-\tfrac19=\tfrac1{18}=\operatorname{Var}(Y)$,故 $\rho=\dfrac{1/36}{1/18}=\tfrac12$——不弱的正相关,全由支撑形状贡献(密度可是常数!)。
协方差的来源可以纯几何:密度可分离却 $\rho=\tfrac12$——歪支撑独立自动出相关。这也反证了对比②:三角支撑必不独立。
Q5二元正态小题:一条直线代替一次积分

升级款设备寿命建模为二元正态:$\mu_X=5$,$\sigma_X=1$;$\mu_Y=6$,$\sigma_Y=2$;$\rho=0.6$(单位:千小时)。实测主件撑到 $x=7$。

求 $E[Y\mid X{=}7]$、$\operatorname{Var}(Y\mid X{=}7)$,以及 $P[Y>8.4\mid X{=}7]$。

  1. 直线斜率:$\rho\dfrac{\sigma_Y}{\sigma_X}=0.6\times\dfrac{2}{1}=1.2$。
  2. 条件期望(沿直线走):$E[Y\mid X{=}7]=6+1.2\,(7-5)=\mathbf{8.4}$——主件超均值 2 个单位,备件预期抬 2.4。
  3. 条件方差(恒定,不看 $x$):$\sigma_Y^2(1-\rho^2)=4(1-0.36)=\mathbf{2.56}$,条件标准差 $1.6$。
  4. 条件概率:切片仍正态且 $8.4$ 恰是条件均值 → $P[Y>8.4\mid X{=}7]=\mathbf{0.5}$,查表都省了。
二元正态是本区块唯一"免画图"的岛:三个数($\mu$、$\sigma$、$\rho$)进直线公式,零积分。但特权仅限正态——题面没写 bivariate normal,就老老实实回去画区域。

题面信号动作
"joint density … on 0<x<y<…"先画支撑,别的什么都别做;链式不等式就是现成积分限
"P[X+Y<…] / P[Y>2X]"阴影 = 事件∩支撑,二重积分;均匀密度直接面积比
"marginal density of X"压扁:对另一变量积分,内限从图上切片读
"given X = x / conditional"切片归一 $f/f_X$;支撑跟着切片走,别抄原区间
"E[XY] / Cov / ρ"$\iint g\cdot f$;$\operatorname{Cov}=E[XY]-E[X]E[Y]$,先用支撑猜符号
"are X, Y independent?"可分离 × 矩形支撑,两问都过才算;三角支撑直接判否
"bivariate normal, ρ = …"直线 $\mu_Y+\rho\frac{\sigma_Y}{\sigma_X}(x-\mu_X)$ + 恒方差 $\sigma_Y^2(1-\rho^2)$,免积分

高频坑二 · 换序要重画图:换积分次序 = 换切法,内外限全部从图上重读(见 Q1:外限从 $0\to\tfrac12$ 变成 $0\to1$)——机械对调上下限必错。

01
Ch27 · Lesson 27 · 课本 p.453

联合矩 = 二重积分

第十二章里对离散联合分布,\(E[g(X,Y)]=\sum_x\sum_y g(x,y)\,p(x,y)\)。到了连续型,把双重求和换成二重积分、把联合概率质量 \(p\) 换成联合密度 \(f\),就是全部改动。

定义:任意函数 \(g(X,Y)\) 的期望 考频 ★★★

课本 p.453 · Lesson 27 开篇

求 \(E[g(X,Y)]\) 就一招:把 \(g(x,y)\) 加权联合密度后,在整个支撑上二重积分。

★ 必背Expectation of a function of X and Y $$E[g(X,Y)]=\iint g(x,y)\,f(x,y)\,dx\,dy$$
🗝️求和换积分,pmf 换 pdf — 与 L12 离散版逐字对应:\(\sum_x\sum_y g\,p\ \to\ \iint g\,f\,dx\,dy\),概念一个没变。

选好积分次序(先 \(dy\) 后 \(dx\) 或反之)后,内外层积分限由联合密度的支撑区域决定——支撑是矩形则上下限是常数,是三角形/带状则内层限含外层变量。

常用特例:直接代 \(g\) 即可 考频 ★★

课本 p.453 · 由定义特例化

令 \(g=xy\) 得 \(E[XY]\)——算协方差 / 相关系数的原料,下一章就用。

Common joint moments $$E[XY]=\iint xy\,f(x,y)\,dx\,dy,\qquad E[X^{m}Y^{n}]=\iint x^{m}y^{n}\,f(x,y)\,dx\,dy$$

令 \(g=x+y\) 得 \(E[X+Y]\);令 \(g=x\) 就退回单变量 \(E[X]=\iint x\,f\,dx\,dy\)。

矩形支撑 \(0\le x\le1,\,0\le y\le1\):内外限都是常数 \(\int_0^1\!\int_0^1\)
三角形支撑 \(0\le x\le y\le1\):内层限含外层变量 \(\int_0^1\!\int_0^{y}\)
⚠️头号失分点 — 积分限:支撑不是矩形时,内层积分限必须写成外层变量的函数(如 \(0\le x\le y\) 则内层 \(x\) 从 0 到 \(y\)),不能一律写 0 到 1。先画支撑区域再落笔。
02
最省事的性质

线性:和的期望 = 期望的和

不管 \(X,Y\) 独立与否、也不管取什么分布,期望永远可加。这是本章最好用、最不会错的一条,能省掉一次二重积分。

期望的线性(Linearity)考频 ★★★

课本 p.453 · 恒成立

和的期望 = 期望的和,恒成立——用它绕开二重积分,是最不会错的一条。

★ 必背Linearity of expectation $$E[X+Y]=E[X]+E[Y]\qquad E[aX+bY]=a\,E[X]+b\,E[Y]$$
🗝️加法随便拆,乘法看独立 — 线性与独立无关;\(E[XY]\) 要拆,必须先验独立(下一节)。

求 \(E[X+Y]\) 时可以直接对联合密度积 \(\iint (x+y)f\,dx\,dy\),也可以先各自求边际期望 \(E[X],E[Y]\) 再相加——两条路结果必然一致,挑好算的走。

💡要点:线性对加法永远成立,和独立无关;但乘法 \(E[XY]\) 一般 \(\neq E[X]E[Y]\),要拆必须先确认独立(下一节)。别把两者混为一谈。
03
Ch27 · 核心陷阱

独立时:乘积期望可拆

当且仅当 \(X,Y\) 独立——联合密度能拆成两个边际的乘积 \(f(x,y)=f_X(x)f_Y(y)\)、且支撑是矩形(\(x\) 的范围不牵扯 \(y\))——乘积的期望才等于期望的乘积。

独立 \(\Rightarrow\) 乘积期望拆开 考频 ★★★

课本 p.453 · Example 27C 前

独立时,乘积的期望 = 期望的乘积——二重积分自动拆成两个一重积分,不必真去积二维。

★ 必背Independence factors the product moment $$X\perp Y\;\Longrightarrow\;E[g(X)\,h(Y)]=E[g(X)]\,E[h(Y)]$$

特例:\(E[XY]=E[X]\,E[Y]\)、\(E[XY^2]=E[X]\,E[Y^2]\)。

如何认独立:① 密度可因式分解为"只含 \(x\)"×"只含 \(y\)";且 ② 支撑是矩形(每个变量的范围是固定常数,不随另一个变量变)。两条缺一不可。

展开 · 为什么独立就能拆

独立时 \(f(x,y)=f_X(x)f_Y(y)\) 且支撑是矩形,于是 \(\iint g(x)h(y)\,f_X(x)f_Y(y)\,dx\,dy=\Big(\int g(x)f_X(x)\,dx\Big)\Big(\int h(y)f_Y(y)\,dy\Big)\)——内外层积分互不牵扯,直接分离成两个一重积分之积。支撑一旦耦合,内层积分限含外层变量,这步分离就做不了。

⚠️最大陷阱 — 只有独立才能拆 \(E[XY]\):若支撑写成 \(0\le x\le y\le1\)(\(x\) 的上限是 \(y\)),即便密度形式看着能拆,也不独立,\(E[XY]\neq E[X]E[Y]\),必须老实二重积分。反例见下面 Example 27B(不独立)对比 27C(独立)。
04
课本例题精解

三道课本 Example 走一遍

课本用三道紧挨着的例题打通"套公式 → 认独立 → 用拆分捷径"。27B(不独立)与 27C(独立)刻意对照,务必对比看。

Example 27A课本 p.453 · 线性 + 二重积分

联合密度 \(f(x,y)=1.2\,(x^2+y)\),\(0\le x\le1,\ 0\le y\le1\)。求 \(E[X+Y]\)。

直接套定义,\(g=x+y\): $$E[X+Y]=\int_0^1\!\int_0^1 1.2\,(x+y)(x^2+y)\,dy\,dx=\int_0^1\!\int_0^1 1.2\,(x^3+xy+x^2y+y^2)\,dy\,dx.$$ 先对 \(y\) 积:\(\int_0^1(\cdots)\,dy=x^3+\tfrac12 x+\tfrac12 x^2+\tfrac13\);再对 \(x\) 积:\(1.2\big(\tfrac14+\tfrac14+\tfrac16+\tfrac13\big)=1.2\).

E[X+Y] = 1.2

步骤用的公式 / 定理出处
① 套联合矩定义,取 \(g(x,y)=x+y\)\(E[g(X,Y)]=\displaystyle\iint g(x,y)\,f(x,y)\,dx\,dy\)本章
② 支撑是单位正方形 → 定积分限\(\int_0^1\!\int_0^1(\cdots)\,dy\,dx\)L24 联合连续 ↗
③ 先 \(y\) 后 \(x\) 逐层积分\(1.2\big(\tfrac14+\tfrac14+\tfrac16+\tfrac13\big)=1.2\)L24 联合连续 ↗
Example 27B课本 p.453 · 三角形支撑,不独立

联合密度 \(f(x,y)=2\),\(0\le x\le y\le1\)(三角形支撑)。求 \(E[XY^2]\)。

支撑非矩形,\(x\) 从 0 积到 \(y\),不能拆: $$E[XY^2]=\int_0^1\!\int_0^{y} 2\,xy^2\,dx\,dy=\int_0^1 2y^2\cdot\frac{x^2}{2}\Big|_0^{y}\,dy=\int_0^1 y^4\,dy=\frac15.$$

E[XY²] = 1/5
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① 看支撑:三角形(\(x\) 上限含 \(y\))→ 不独立,矩不能拆独立需密度可分解且支撑为矩形L24 独立判据 ↗
② 联合矩定义 + 按区域定限:内层 \(x\) 从 0 积到 \(y\)\(E[XY^2]=\int_0^1\!\int_0^{y}2\,xy^2\,dx\,dy\)本章 · L24 联合连续 ↗
③ 逐层积分:先 \(x\) 后 \(y\)\(\int_0^1 y^4\,dy=\tfrac15\)L24 联合连续 ↗
Example 27C课本 p.453–454 · 矩形支撑,独立

联合密度 \(f(x,y)=1\),\(0\le x\le1,\ 0\le y\le1\)(单位正方形上的均匀分布)。求 \(E[XY^2]\)。

密度可因式分解、支撑是矩形 \(\Rightarrow\) 独立,直接拆: $$E[XY^2]=E[X]\,E[Y^2]=\Big(\tfrac12\Big)\Big(\tfrac13\Big)=\frac16.$$ 对比 27B:同样问 \(E[XY^2]\),换成三角形支撑就不能这么拆。

E[XY²] = 1/6
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① 密度可分解 + 矩形支撑 → 独立\(f(x,y)=1=f_X(x)\,f_Y(y)\),支撑 \([0,1]^2\)L24 独立判据 ↗
② 独立 ⇒ 矩直接拆开\(E[XY^2]=E[X]\,E[Y^2]\)本章
③ 各算单变量矩(均匀 \([0,1]\))\(E[X]=\tfrac12,\ \ E[Y^2]=\int_0^1 y^2\,dy=\tfrac13\)L15 均匀 ↗ · L07 期望 ↗
💡27B vs 27C 的教训:能否用 \(E[XY^2]=E[X]E[Y^2]\) 这条捷径,只看独立性,不看密度长得像不像。支撑一旦是三角形(\(x\) 上限含 \(y\)),先默认不独立,老实二重积分。

直觉 · 拓展

换个角度看懂它

💡 直觉例子:离散 ↔ 连续,一张对照表

把 L12 的骰子换成飞镖靶:离散是"对每个格子点,拿 \(g\) 值 × 该点概率,逐点加总";连续是"对靶面每一小块,拿 \(g\) 值 × 密度 × 面积微元,把整个靶面扫一遍"——"扫一遍"就是二重积分。方法零新意,只是记号换装:

对象离散(L12)连续(本章)
权重联合 pmf \(p(x,y)\)联合 pdf \(f(x,y)\)
汇总方式\(\sum_x\sum_y\)\(\displaystyle\iint\cdots\,dx\,dy\)
联合矩\(\sum\sum g(x,y)\,p(x,y)\)\(\iint g(x,y)\,f(x,y)\,dx\,dy\)
取值范围枚举取值点支撑区域 → 决定积分限
逐点加总:离散(L12 骰子)是格子点,每点拿 \(g\times p\) 相加
扫一遍:连续把靶面切成小块 \(dx\,dy\),每块贡献 \(g\cdot f\,dx\,dy\)(27A/27C 的单位正方形)

再一个手感:同一场台风下两张保单的赔付 \(X,Y\) 高度相关,但"总赔付的期望"仍 = 各自期望之和(线性不问相关);而"赔付乘积的期望" \(E[XY]\)(下一章协方差的原料)就必须看依赖结构了——加法与乘法的分野,正是本章两条性质的分野。

🎯 SOA 题型套路

  1. 积分限随支撑变:题面一句 "for \(0<x<y<1\)" 就是全部机关——先画支撑,内层限写成外层变量的函数,九成失分在这。
  2. 先找捷径再硬算:见 \(E[aX+bY]\) 直接线性拆开;见 \(E[XY^k]\) 先验独立(密度可因式分解 + 矩形支撑),独立则一维积分 × 一维积分。
  3. 题面的 insurance loss / time-to-failure 联合密度都是包装,剥掉外衣全是同一句话:对 \(g\,f\) 二重积分。

巩固

易错点 + 自测

⚠️ 高频易错点

  • 积分限套错:非矩形支撑内层限必须含外层变量,先画区域再定限。
  • 乱拆 \(E[XY]\):未确认独立就写 \(E[X]E[Y]\)——只有独立才成立。
  • 把线性也当成需要独立:\(E[X+Y]=E[X]+E[Y]\) 恒成立,别多此一举验独立。
  • 积分次序与被积函数错配:先 \(dx\) 就用 \(x\) 的限,别把 \(dx\)/\(dy\) 限写反。
  • 密度常数没归一:题给密度须满足 \(\iint f=1\),代错常数结果全歪。

📝 课本自测(做完对答案)

先自己做,再翻课本 Solutions 核对(p.455 起)。

  • 练 27.1可因式分解密度,判断 \(E[X^2Y^3]=E[X^2]E[Y^3]\) 等命题真伪。
  • 练 27.3给分段联合密度,求 \(E[X]\)(保险事故年龄)。
  • 练 27.6/27.7两部件系统:非矩形支撑上求 \(E[Y]\)、运行时间期望。
  • 练 27.24由联合密度求边际密度,再算 \(\mathrm{Var}(Y)\)。

🔎 60 秒快测(口算)

  • \(X,Y\) 独立且都 \(\sim U(0,1)\),求 \(E[XY]\)。
  • \(f(x,y)=2\) 在 \(0\le x\le y\le1\),写出 \(E[XY^2]\) 的正确积分限。
  • 已知 \(E[X]=3,\ E[Y]=5\),求 \(E[2X+Y]\)(不知是否独立)。
✅参考答案:① \(E[X]E[Y]=\tfrac12\cdot\tfrac12=\tfrac14\)(独立可拆);② \(\int_0^1\!\int_0^{y}2xy^2\,dx\,dy\)(内层 \(x\):0→\(y\));③ \(2\cdot3+5=11\)(线性,不需独立)。

公式速查卡 · 连续型联合矩

定义 / 特例

\(E[g(X,Y)]=\iint g(x,y)f(x,y)\,dx\,dy\)

\(E[XY]=\iint xy\,f\,dx\,dy\)

\(E[X^mY^n]=\iint x^m y^n f\,dx\,dy\)

积分限由支撑定;非矩形内层含外层变量

线性 / 独立

线性(恒成立):\(E[aX+bY]=aE[X]+bE[Y]\)

独立才拆:\(X\perp Y\Rightarrow E[XY]=E[X]E[Y]\)

更一般:\(E[g(X)h(Y)]=E[g(X)]E[h(Y)]\)

三角支撑 \(\Rightarrow\) 不独立,老实二重积分

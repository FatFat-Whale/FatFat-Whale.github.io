核心知识点及考题策略

先扫一遍公式卡热身(带「推」标的悬停看推导);学完回来对照「考点 + 套路」自检,这一节全拿下才算过关。

离散 / pmf

\(p(x)=P[X=x]\)

\(p(x)\ge 0,\;\sum p(x)=1\)

\(F(x)=\sum_{t\le x}p(t)\)(阶梯,跳高 \(=p(x)\))

\(P[X=x]>0\)

连续 / pdf

\(f(x)=F'(x)\ge 0,\;\int_{-\infty}^{\infty}f=1\)

\(P[a\le X\le b]=\int_a^b f\,dx\)(面积)

\(P[X=x]=0\);\(f\) 值可 \(>1\)

\(\le\) 与 \(<\) 可互换

分布函数 cdf

\(F(x)=P[X\le x]\)

非减 · 右连续 · \(F(-\infty)=0,\,F(\infty)=1\)

\(P[u<X\le v]=F(v)-F(u)\)

生存函数

\(S(x)=P[X>x]=1-F(x)\)

单调递减 · \(S(-\infty)=1,\,S(\infty)=0\)

"至少 / 大于"类首选

🎯 本章核心考点

  • 分清离散(pmf \(p(x)=P[X=x]\),求和)与连续(pdf \(f(x)\),积分 \(\int_a^b f\,dx\))两类随机变量,会算区间概率。
  • 用分布函数 \(F(x)=P[X\le x]\) 求区间概率 \(F(b)-F(a)\),由 \(F'(x)=f(x)\) 还原 pdf;"至少 / 大于"类用 \(S(x)=1-F(x)\)。
  • 记牢三条易混:连续下 \(P[X=x]=0\)、\(F\) 是 \(\le\) 不是 \(<\)、pdf 值不是概率(可 \(>1\))。
  • 认得 §5.1 四个保险条款(免赔额 / 给付限额 / 共保 / 通胀)并写出赔付表达式。

🧭 题型套路(SOA 真题)

  1. 给分段 \(F\) 求区间概率:直接 \(F(b)-F(a)\),先看清端点落在哪一段(例 5A 里 \(F(2.5)=1\));离散/混合分布盯紧 \(\le\) 与 \(<\)——差一个跳点 \(p(x)\)。
  2. 给含常数 \(c\) 的密度:必是"先 \(\int f=1\) 解 \(c\),再积分求区间"两步题。
  3. 保险语境:损失额 \(X\) 常是混合型(\(X=0\) 有质量点),"损失超过 \(x\)"直接读作 \(S(x)\)——这两件事到 L06 会和免赔额一起变成大题。

随机变量 — 定义与两大类

掷骰子的点数、样本中的次品数、保单提交的索赔数、一笔理赔的金额——它们都是"把随机试验的结果翻译成一个实数"的规则。这个规则就是随机变量。

定义:随机变量(random variable)考频 ★★

课本 p.85

随机变量是从样本空间(事件空间)到实数集的一个函数——把每个结果映射成一个实数。

习惯用大写字母(常取字母表末尾 \(X,Y,Z\))记随机变量,用小写 \(x\) 记它的取值。"点概率"记 \(P[X=x]\),简写 \(p(x)\) 或 \(p_x\);"在某范围"记 \(P[u<X\le v]\)。

骰子点数是 \(1,2,3,4,5,6\);样本次品数、保单索赔数都是非负整数;理赔金额是非负实数。

离散 Discrete连续 Continuous
取值可数的一串点(如整数)填满一段区间里的所有实数
概率长相集中在点上,每点 \(P[X=x]>0\)摊在范围上,单点 \(P[X=x]=0\)
工具pmf \(p(x)\),求和pdf \(f(x)\),积分
cdf 形态阶梯,跳高 \(=p(x)\)光滑上升的 S 型
例子骰子点数、索赔个数身高、理赔金额
💡还有第三种:课本提到部分随机变量是混合型(mixed)——既有集中在点上的概率(如"零索赔"这一点有正概率),又有摊在范围上的概率。保险损失常是这种,后面会遇到。
02
离散 · 课本 p.86

概率函数 / 概率质量函数 pmf

离散随机变量由"每个点占多少概率"完全刻画,这张"点→概率"的表就是概率函数。

概率质量函数(probability mass function, pmf)考频 ★★★

课本 p.86 · 又叫 probability function

离散分布就是一张"点 → 概率"的表:每点非负,全部加起来正好是 1。

pmf of a discrete random variable $$p(x)=P[X=x]$$

两条合法性要求(整块记):

properties $$p(x)\ge 0\qquad\text{且}\qquad \sum_{\text{all }x} p(x)=1$$

每个点的概率非负,所有点的概率加起来正好是 1。掷均匀骰子 \(p(x)=\tfrac16\)(\(x=1,\dots,6\)),六项相加 \(=1\)。

离散 pmf:概率是一根根柱子,高度 \(=p(x)\),柱高之和 \(=1\)
离散 cdf:\(F\) 是阶梯,在每个点跳一跳,跳高 \(=p(x)\)(课本图 5.2)
📝离散 cdf 由 pmf 累加:\(\;F(x)=\displaystyle\sum_{t\le x}p(t)\;\)——把 \(x\) 及以下所有点的概率加起来。骰子 \(F(1.88)=P[N\le 1]=\tfrac16\)(课本 Example 5B)。
03
连续 · 课本 p.87

概率密度函数 pdf

连续随机变量没有"点概率"可加,概率靠一条密度曲线下方的面积表示——想要区间概率就积分。

概率密度函数(probability density function, pdf)考频 ★★★

课本 p.87

连续概率是曲线下的面积——要区间概率就积分,单点面积为 0。

★ 必背interval probability = area under the curve $$P[a\le X\le b]=\int_a^b f(x)\,dx$$
🗝️离散数柱子,连续量面积 — pmf 把概率堆在点上求和,pdf 把概率摊在区间上积分;问法一样,动作换一个。

密度 \(f(x)\) 是分布函数的导数,\(f(x)=\dfrac{dF(x)}{dx}\);反过来概率是密度在区间上的积分。

两条合法性要求:

properties $$f(x)\ge 0\qquad\text{且}\qquad \int_{-\infty}^{\infty} f(x)\,dx=1$$

连续下单点概率为 0:\(P[X=x]=0\),所以 \(\le\) 与 \(<\) 可随意互换,\(P[a\le X\le b]=P[a<X<b]\)。这对离散不成立。

连续 pdf:\(P[a\le X\le b]\) = 曲线下 \([a,b]\) 段的阴影面积
连续 cdf:\(F\) 是光滑上升的 S 型曲线,从 0 升到 1,\(F'(x)=f(x)\)
⚠️密度 ≠ 概率:\(f(x)\) 的值可以大于 1(它是"每单位长度的概率",不是概率本身)!只有它在区间上的积分(面积)才是概率,且总面积恰为 1。见课本 p.88 关于"proper"密度的说明。

随机变量 \(X\) 的密度为 \(f(x)=cx^2\)(\(1\le x\le 4\),其余为 0)。求常数 \(c\),并计算 \(P[2\le X\le 3]\)。

① 定 \(c\):全区间积分等于 1。\(\displaystyle\int_1^4 cx^2\,dx=c\cdot\frac{4^3-1^3}{3}=c\cdot\frac{63}{3}=21c=1\),解得 \(c=\dfrac{1}{21}\)。
② 求区间概率:\(\displaystyle P[2\le X\le 3]=\int_2^3\frac{x^2}{21}\,dx=\frac{3^3-2^3}{3(21)}=\frac{27-8}{63}=\frac{19}{63}\)。

c = 1/21　P[2≤X≤3] = 19/63 ≈ 0.302
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① pdf 合法性(总面积 = 1)反解常数 \(c\)\(\displaystyle\int_{-\infty}^{\infty}f(x)\,dx=1\ \Rightarrow\ 21c=1,\ c=\tfrac{1}{21}\)本章
② 区间概率 = 密度在区间上积分\(\displaystyle P[a\le X\le b]=\int_a^b f(x)\,dx=\int_2^3\frac{x^2}{21}\,dx=\frac{19}{63}\)本章
💡方法套路:给"含未知常数的密度"→ 先用 \(\int f=1\) 反解常数,再积分求所求区间。课本 Example 5C(\(f(x)=\tfrac{1}{(1+x)^2}\),\(x\ge0\))也是同一路子,答案 \(P[2<X<5]=\tfrac16\)。
04
离散连续通用 · 课本 p.85–87

分布函数 cdf 与生存函数

分布函数 \(F(x)\) 对离散、连续一视同仁——"知道 \(F\) 就等于知道这个随机变量的一切"。它是刷题时求各种范围概率的统一入口。

累积分布函数(cumulative distribution function, cdf)考频 ★★★

课本 p.85 · 式定义

\(F(x)\) 累住"到 \(x\) 为止"的全部概率;任何区间概率都是两个 \(F\) 相减。

★ 必背cdf 与区间概率 $$F(x)=P[X\le x]\qquad P[u<X\le v]=F(v)-F(u)$$
🗝️F 管"不超过",S 管"超过" — 俩加起来恒等于 1;看到 at least / greater than,先想 \(S\)。

四条必备性质(整块记):

properties of F $$\text{非减(nondecreasing)}\quad\text{右连续(right-continuous)}\quad F(-\infty)=0\quad F(\infty)=1$$

连续时 \(F'(x)=f(x)\);离散时 \(F(x)=\displaystyle\sum_{t\le x}p(t)\)。注意范围符号:\(F(x)\) 用的是 \(\le\) 不是 \(<\)。对离散随机变量这一步差一个跳点,答案就错。

展开 · 为什么离散的 F 在跳点"含端点"

\(F(x)=P[X\le x]\) 把 \(x\) 本点也算进去,所以阶梯在每个质量点跳 \(p(x)\)、且跳完的值算在该点上(右连续)。骰子:\(F\) 在 \(x=1\) 跳到 \(\tfrac16\),之后 \(F(1.88)\) 仍 \(=\tfrac16\),直到 \(x=2\) 再跳。求 \(P[X<x]\) 时要用跳前的值,与 \(F(x)\) 差一个 \(p(x)\)。

生存函数(survival function)考频 ★★

保险常用 · 与 \(F\) 互补

"活过 \(x\) / 损失超过 \(x\)"的概率;精算里描述寿命、损失尾部全靠它。

★ 必背survival function $$S(x)=P[X>x]=1-F(x)$$

求"至少"、"大于"类问题用它最省事。\(S(-\infty)=1,\;S(\infty)=0\),随 \(x\) 单调递减。

例 5A课本 p.85 · cdf 求区间

已知 \(F(x)=0\)(\(x<0\))、\(F(x)=\dfrac{x^2}{4}\)(\(0\le x\le 2\))、\(F(x)=1\)(\(x>2\))。求 \(P[1.5<X\le 2.5]\)。

直接用 \(F(v)-F(u)\):\(\;F(2.5)=1\)(因 \(2.5>2\)),\(F(1.5)=\dfrac{1.5^2}{4}=\dfrac{9}{16}\)。
故 \(P[1.5<X\le 2.5]=F(2.5)-F(1.5)=1-\dfrac{9}{16}=\dfrac{7}{16}\)。

P[1.5<X≤2.5] = 7/16
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① 端点各自落进哪一段,读出 \(F\) 值\(2.5>2\Rightarrow F(2.5)=1;\ \ F(1.5)=\dfrac{1.5^2}{4}=\dfrac{9}{16}\)本章
② 区间概率 = 两个 \(F\) 相减\(P[u<X\le v]=F(v)-F(u)=1-\dfrac{9}{16}=\dfrac{7}{16}\)本章

💡 直觉例子:密度为什么能大于 1

设 \(X\) 均匀分布在 \([0,\tfrac12]\) 上,密度 \(f(x)=2\)——区间只有 0.5 长,密度不到 2,面积凑不满 1。密度是"每单位长度的概率",好比人口密度:每平方公里 2 万人不代表总人口 2 万,还得乘上面积。例 5D 那类"先用 \(\int f=1\) 定 \(c\)"的题,本质就是在校准这把"密度尺"。

密度尺:\(f(x)=2\) 越过 1 不违法,概率是面积 \(2\times\tfrac12=1\)
楼梯与坡道:离散 cdf 跳级上楼,连续 cdf 平滑爬坡,顶都是 1

再一个:楼梯与坡道。离散 cdf 是楼梯——每到一个质量点跳一级,台阶高度 = 该点概率;连续 cdf 是坡道——平滑爬升,坡度(导数)就是密度。两条路爬到顶都是 1。

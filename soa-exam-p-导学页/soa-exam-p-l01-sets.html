Ch1 · Lesson 1 · 课本 p.1–4

Sets — 集合论

概率是给"不确定事件"赋一个 0~1 的数。要操作"且 / 或 / 非"这些事件组合,先把事件当成集合来处理。

核心概念:集合与三大运算 考频 ★

课本 p.1 · 视频 知识点讲解 (1.2x)

把事件当集合处理:\(\cup\) 是「或」、\(\cap\) 是「且」、\('\) 是「非」,全集 \(\Omega\) 装下所有结果。

Union / Intersection / Complement \( A\cup B\):并,\(x\) 在 \(A\) 或 \(B\) 中  ·  \( A\cap B\):交,\(x\) 同时在 \(A\) 与 \(B\) 中  ·  \( A'\)(也写 \(A^{c},\bar A\)):补,\(\Omega\) 中不属于 \(A\) 的部分

集合(set):对象的 collection,里面的对象叫成员(members)。两个特殊集合:全集 \(\Omega\)(entire space),所有成员都来自它;空集 \(\varnothing\)(empty set),没有任何成员。

💡记号约定:课本用 \(A'\) 表示补集(与 SOA 样题一致),但 SOA 的官方解答常用 \(A^{c}\)。两者同义,看到别懵。

Venn 图:三个必背阴影(对应课本 Fig 1.1–1.3)

\(A\cup B\) · 并(两圆全染)
\(A\cap B\) · 交(只染重叠)
\((A\cup B)'\) · 补(只染外部)
🎯若 \(A\cap B=\varnothing\),称两集合互斥(mutually exclusive)——两圆不相交。这是后面概率公理的关键场景。

集合运算律(必背)考频 ★★

课本 p.1 · Table 1.1

三条运算律里 De Morgan 最常考:取补时,∪ 和 ∩ 必须互换。

★ 必背De Morgan 律(补的分配)— 高频考点 $$(A\cup B)' = A'\cap B'\qquad (A\cap B)' = A'\cup B'$$
🗝️撇进括号,符号翻面 — 「非(A 或 B)」=「非 A 且 非 B」:撇分给每个字母,∪↔∩ 对调。
结合律 Associative $$(A\cup B)\cup C = A\cup(B\cup C)\qquad (A\cap B)\cap C = A\cap(B\cap C)$$
分配律 Distributive $$A\cup(B\cap C)=(A\cup B)\cap(A\cup C)\qquad A\cap(B\cup C)=(A\cap B)\cup(A\cap C)$$
⚠️易错:De Morgan 取补时,∪ 和 ∩ 要互换,不是原样加撇。"非(A或B)= 非A 且 非B"。
例 1A课本 p.3

化简 \((A\cup B)\cap(A\cup B')\)。

用分配律提出 \(A\):\((A\cup B)\cap(A\cup B') = A\cup(B\cap B')\)。
因为 \(B\) 与 \(B'\) 互斥,\(B\cap B'=\varnothing\),于是 \(=A\cup\varnothing\)。

= A

概率三公理 → 常用性质 考频 ★★

课本 p.3

全部概率规则都从三公理长出来;解题最常用的一条是 \(P[A']=1-P[A]\)。

三公理:① 任意集合 \(P[A]\ge 0\);② \(P[\Omega]=1\);③ 可数个互斥集合的并,其概率 = 各概率之和。

由公理推出的常用性质 $$P[A]\le 1\qquad P[A']=1-P[A]\qquad P[A\cap B]\le P[A]$$

把"非"翻译成 \(P[A']=1-P[A]\) 是解题时最常用的一步(题目给"不发生"的比例时)。

展开 · 为什么 \(P[A']=1-P[A]\)

\(A\) 与 \(A'\) 互斥且 \(A\cup A'=\Omega\),由公理③:\(P[A]+P[A']=P[\Omega]=1\),移项即得。另两条性质同样由公理③加"子集概率不超过全集"推出。

容斥公式(本课重点)考频 ★★★

课本 p.3–4 · 式 (1.1)(1.3)

直接加 \(P[A]+P[B]\) 会把交集数两遍——减掉一次,才是并集的概率。

★ 必背两集合 Inclusion–Exclusion $$P[A\cup B]=P[A]+P[B]-P[A\cap B]$$ $$\Longleftrightarrow\quad P[A\cap B]=P[A]+P[B]-P[A\cup B]$$
★ 必背三集合(考试最多到三集合) $$P[A\cup B\cup C]=P[A]+P[B]+P[C]-P[A\cap B]-P[A\cap C]-P[B\cap C]+P[A\cap B\cap C]$$
🗝️加单、减双、加三 — 加单个、减两两交、加回三重交,正负交替;谁被多数了一遍,就把谁退回去。

看 \(A\cup B\) 由三块互斥区域组成:只在 \(A\)、只在 \(B\)、以及交集 \(A\cap B\)——直接相加就把交集数了两遍。

展开 · 一般形式(式 1.3)怎么记

规律:加单个、减两两交、加三个交……正负交替;\(n\) 集合时,第 \(k\) 层交集前的符号是 \((-1)^{k+1}\)。考试最多到三集合,记住节奏即可。

并 \(A\cup B\)(或)交 \(A\cap B\)(且)
读题信号at least one / either … or …(至少一个)both … and …(同时;都)
Venn 图两圆全染只染重叠
概率工具容斥:\(P[A]+P[B]-P[A\cap B]\)反解容斥:\(P[A]+P[B]-P[A\cup B]\)
取补(De Morgan)\((A\cup B)'=A'\cap B'\)(都不发生)\((A\cap B)'=A'\cup B'\)(至少缺一个)
例 1B课本 p.3 · 两集合

设 \(A\)=参加野餐,\(B\)=参加高尔夫/网球。已知不参加野餐 35%、不参加球局 80%、两者都不参加 25%。问两者都参加的比例?

先把"不"翻成补:
\(P[A]=1-0.35=0.65\);\(P[B]=1-0.80=0.20\);
\(P[A\cup B]=1-P[(A\cup B)']=1-0.25=0.75\)。
再套交集容斥:\(P[A\cap B]=0.65+0.20-0.75\)。

= 0.10
例 1C课本 p.4 · 三集合

寿险/健康/伤残三种保障。10% 三种都没有;单项持有 25% / 75% / 20%;两两同时持有 15% / 12% / 5%。求三种都持有的比例 \(x\)?

三者并 \(=1-0.10=0.90\)。代入三集合容斥:
\(0.90=0.25+0.75+0.20-0.15-0.12-0.05+x\)
\(0.90=0.88+x\)。

x = 0.02
📝本课题型:课本明确说——本课考题基本都在考 2 或 3 个集合的容斥。剩下的功夫在"把文字翻成 \(P[\cdot]\) 的哪一项"。

直觉 · 拓展

换个角度看懂它

💡 直觉例子:数人头,别数两遍

班里 30 人:20 人会 Python(\(A\)),15 人会 R(\(B\)),10 人两样都会。会至少一样的有几人?直接 \(20+15=35\) 比全班还多——两样都会的 10 人在两份名单里各被点了一次名。退回多点的那次:\(20+15-10=25\) 人。这就是容斥「减交集」的全部道理。

别数两遍:两份名单直接相加,中间 10 人被算了两次,必须退回一次
人头版容斥:四块加起来正好 30;\(20+15-10=25\) 人至少会一样,圈外 5 人都不会

顺手可得「都不会」的人数:\(30-25=5\),正是 \(P[(A\cup B)']=1-P[A\cup B]\) 的人头版。De Morgan 也有同款直觉:「不(会 Python 或 会 R)」=「不会 Python 且 不会 R」——否定「至少有一个」,等于「一个都没有」,所以取补时 ∪ 变 ∩。

🎯 SOA 题型套路

  1. 本章真题几乎全是 2–3 集合容斥 + 补集翻译:题面给「都不持有 \(x\%\)」,先转 \(P[A\cup B]=1-x\) 再套容斥。
  2. 保险语境信号词:policyholder(保单持有人)、has coverage(持有某险种)、neither … nor …(= 补集之交 \((A\cup B)'\))、at least one(= 并)。
  3. 陷阱:「only A / 只有 A」是 \(A\cap B'\),不是 \(A\);三集合题先画 Venn 分 8 块,从最里层(三重交)向外填数,填完再读题问的是哪几块。

巩固

易错点 + 自测(Sets)

⚠️ 高频易错点

  • 互斥 ≠ 独立:互斥是 \(A\cap B=\varnothing\)(集合层面);独立是另一个概念(后面才学),别混。
  • 容斥忘减交集:\(P[A\cup B]\) 一定要 \(-P[A\cap B]\),否则重复计数。
  • De Morgan 不换符号:取补后 ∪↔∩ 必须互换。
  • "不"要转补集:题给"不发生"比例时,先用 \(P[A']=1-P[A]\)。

📝 课本自测(做完对答案)

先自己做,再翻课本 Solutions 核对。

  • 练 1.1已知 \(P[X],P[Y]\),判断 \(P[X\cap Y]\) 的不等式范围。
  • 练 1.7容斥求"只拥有其一"(车 or 房,但不都有)。
  • 练 1.10三种保险的容斥(求"有 A 但没 B")。
  • 练 1.35三风险因子的人数统计(三集合容斥经典)。

🔎 60 秒快测(口算)

  • 已知 \(P[A]=0.6,\ P[B]=0.5,\ P[A\cup B]=0.8\),求 \(P[A\cap B]\)。
  • "既不抽烟也不喝酒"占 30%,如何用 \(P[\,\cdot\,']\) 表示"抽烟或喝酒"?
  • 用 De Morgan 把 \((A\cap B)'\) 展开成补集运算。
✅参考答案:① \(0.6+0.5-0.8=0.3\);② \(1-0.30=0.70\);③ \((A\cap B)'=A'\cup B'\)。

集合运算

\((A\cup B)'=A'\cap B'\)

\((A\cap B)'=A'\cup B'\)

分配律 \(A\cup(B\cap C)=(A\cup B)\cap(A\cup C)\)

互斥:\(A\cap B=\varnothing\)

概率 / 容斥

\(P[A']=1-P[A]\)

\(P[A\cup B]=P[A]+P[B]-P[A\cap B]\)

\(P[A\cup B\cup C]=\sum P-\sum P_{两两}+P_{三}\)

公理:\(P[\Omega]=1,\ P[A]\ge 0\)

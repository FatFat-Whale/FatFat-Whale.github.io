01
Ch24 · Lesson 24 · 课本 p.427

联合密度函数 — 定义与性质

离散时用联合概率函数;连续时概率被"抹平",单点概率为 0,只能谈区域上的概率——由联合密度函数 \(f(x,y)\) 在区域上积分得到。密度可以任意指定,只要满足两条约束。

联合密度函数(joint p.d.f.)的两条性质 考频 ★★★

课本 p.427

密度处处非负、全平面积分为 1——第二条就是反解未知常数 \(k\) 的方程。

★ 必背Two defining properties $$f(x,y)\ge 0\qquad\text{且}\qquad \iint_{\mathbb{R}^2} f(x,y)\,dy\,dx=1$$

把 \(f\) 想成"单位面积上的概率密度":总概率 = 密度曲面下的总体积 = 1。第二条常用来反解未知常数(令积分 = 1 解出 \(k\),见自测 24.1/24.2)。

展开 · 反解 k 的固定套路

设 \(f=k\,g(x,y)\)(支撑内)。先算 \(\iint_{\text{支撑}} g\,dA\)(限按支撑设),令 \(k\iint g=1\) 解出 \(k\);最后检查解出的 \(f\ge0\)。

区域概率 = 二重积分 考频 ★★★

课本 p.427

区域上的概率 = 密度在该区域上的二重积分;核心动作是把区域翻译成积分限。

Probability over a rectangle $$P[a<X\le b,\ c<Y\le d]=\int_a^b\!\!\int_c^d f(x,y)\,dy\,dx$$

更一般地,任意区域 \(R\) 上:\(\displaystyle P[(X,Y)\in R]=\iint_R f(x,y)\,dA\)。核心动作是把 \(R\) 与支撑的交集翻译成积分限。

支撑上 \(\iint f=1\)(密度曲面下总体积 = 1)
\(P[(X,Y)\in R]=\iint_R f\,dA\)(黄=积分区域)
💡直觉:联合密度是一张盖在平面上的"帐篷",总体积 = 1;要某区域的概率,就量帐篷在那块地皮上方的体积。所有难点都在"把地皮描述成积分限"。
02
本章命根子

按支撑设积分限

同一个密度,概率对不对全看积分限对不对。矩形支撑内外层都是常数;三角形 / 斜边有界支撑里,内层限会随外层变量变动——这是本章最容易错、也最常考的地方。

两步走:先画支撑 ∩ 区域,再定限 考频 ★★★

课本 p.427–428

先画支撑与目标区域的交集,再定限:外层定常数范围,内层随外层变量走。

★ 必背Non-rectangular support ⇒ inner limits depend on outer var $$P[(X,Y)\in R]=\int_{x_{\min}}^{x_{\max}}\!\!\int_{y_{\text{low}}(x)}^{y_{\text{high}}(x)} f(x,y)\,dy\,dx$$
🗝️外层常数,内层可变 — 内层积分限只能含常数或外层变量;积完还剩内层变量 = 必错。

① 画出支撑(\(f>0\) 的区域)与目标区域 \(R\),取交集;② 选外层变量(如 \(x\)),定它的常数范围;③ 固定该 \(x\),看内层变量(\(y\))从哪到哪——若边界是斜线,内层限就是含 \(x\) 的表达式。

矩形:\(X<0.5,\,Y<0.4\),内外层限都是常数(例 24A)
斜边:\(X+Y>0.8\),补集是小三角(例 24B)
三角形 \(0<x<y<1\):内层 \(y\) 从 \(x\) 到 \(1\)(随 \(x\) 变)
例 24A课本 p.427 · 矩形区域

联合密度 \(f(x,y)=1.2(x^2+y)\),\(0\le x\le1,\ 0\le y\le1\)。求 \(P[X<0.5,\,Y<0.4]\)。

区域是矩形,限都是常数。先对 \(y\) 内层积分:

$$\int_0^{0.5}\!\!\int_0^{0.4}1.2(x^2+y)\,dy\,dx=\int_0^{0.5}1.2\Big(0.4x^2+\tfrac{0.4^2}{2}\Big)dx=\int_0^{0.5}1.2(0.4x^2+0.08)\,dx$$ $$=1.2\Big(0.4\cdot\tfrac{0.5^3}{3}+0.08\cdot0.5\Big)=1.2(0.0166\overline{6}+0.04)=0.068$$ P[X<0.5, Y<0.4] = 0.068
例 24B课本 p.428 · 斜边区域 + 取补技巧

同一 \(f(x,y)=1.2(x^2+y)\),\(0\le x,y\le1\)。求 \(P[X+Y>0.8]\)。

直接算需按斜线 \(x+y=0.8\) 拆成两块积分(繁);更省事是取补:补集 \(X+Y\le0.8\) 是左下小三角,内层 \(y\) 从 \(0\) 到 \(0.8-x\),外层 \(x\) 从 \(0\) 到 \(0.8\):

$$P[X+Y\le0.8]=\int_0^{0.8}\!\!\int_0^{0.8-x}1.2(x^2+y)\,dy\,dx=0.14336$$ $$\Rightarrow\ P[X+Y>0.8]=1-0.14336=0.85664$$ P[X+Y>0.8] = 0.85664
🗝️斜边先取补 — \(X+Y>c\) 这类斜线事件,先看补集是不是一块更好积的三角形,常把两块积分省成一块。
⚠️三大限坑:① 内层限里出现内层自己的变量(必错,积分完还带 \(y\));② 矩形想当然套到斜边区域,漏掉 \(0.8-x\);③ 内外层顺序与 \(dy\,dx\) 不匹配。斜边区域优先考虑取补,常把两块积分省成一块。
03
Ch24 · 对偶

联合累积分布函数 cdf

和一元一样,联合 cdf 是"两个都不超过"的概率;它是密度的积分,密度是它的混合二阶偏导——pdf 与 cdf 互为微积分对偶。

定义与互求 考频 ★★

课本 p.427

给了 \(F\) 求概率直接代值作差;要密度就求两次偏导——别把 cdf 又拿去积分。

Joint c.d.f. $$F(x,y)=P[X\le x,\ Y\le y]=\int_{-\infty}^{x}\!\!\int_{-\infty}^{y} f(s,t)\,dt\,ds$$

反过来,对 \(F\) 求混合二阶偏导就回到密度:

★ 必背Density = mixed 2nd partial of the c.d.f. $$f(x,y)=\frac{\partial^2 F(x,y)}{\partial x\,\partial y}$$

性质:\(F\) 对每个变量都不减,\(F(-\infty,\cdot)=F(\cdot,-\infty)=0\),\(F(\infty,\infty)=1\);边缘 cdf \(F_X(x)=F(x,\infty)\)。给了 \(F\) 求某点概率(如自测 24.5 求 \(F(4,5)\))直接代入即可。

展开 · 为什么求两次偏导就回到密度

微积分基本定理用两次:先对 \(x\) 偏导剥掉外层积分,再对 \(y\) 偏导剥掉内层积分,剩下的正是被积函数 \(f(x,y)\)。

题给联合 pdf \(f\)题给联合 cdf \(F\)
求区域概率二重积分 \(\iint_R f\,dA\)(定限是关键)直接代值 / 作差,别再积分
求另一个\(F(x,y)=\int_{-\infty}^{x}\!\int_{-\infty}^{y}f\,dt\,ds\)(积分)\(f=\dfrac{\partial^2 F}{\partial x\,\partial y}\)(混合二阶偏导)
一句话给密度 → 动积分给 cdf → 动偏导 / 代值

直觉 · 拓展

换个角度看懂它

💡 直觉例子:帐篷、地皮与体积

联合密度是一张盖在 \(xy\) 平面上的帐篷,帐篷下的总体积恰好是 1。求"落在区域 \(R\)"的概率,就是量帐篷在那块地皮上方的体积——二重积分只是量体积的工具。单点、一条线的概率为什么是 0?因为地皮没有面积,上方就没有体积。

帐篷量体积:例 24A 的 \(R:\ x<0.5,\ y<0.4\),\(P\) 就是帐篷在 R 上方那截的体积
地皮没面积就没体积:线 \(y=x\) 的地皮宽度为 0,切出的薄片没有厚度,所以概率为 0

和 L11 离散联合分布对照着记:那里把表格里的格子概率相加,这里把密度积分——"求和 ↔ 积分"一换,整套语言(联合 / 边缘 / 条件 / 独立)原样平移到连续世界。

🎯 SOA 题型套路

  1. 积分限画图先行:真题支撑十有八九是三角形 / 梯形(\(0<x<y<1\)、\(x+y<1\)、\(0<y<x\)),先画支撑再定限,内层限带外层变量。
  2. 反解常数:「what is the value of k?」= 令 \(\iint f=1\)。
  3. 保险外衣:\(X\)=损失、\(Y\)=理赔费用或时间,支撑常写 \(0<y<x\)(费用不超过损失)。
  4. 给 cdf 的题(如 24.5)直接代值作差,别再积分。
  5. 自检:内层积完不该再剩内层变量。

巩固

易错点 + 自测

⚠️ 高频易错点

  • 非矩形支撑设成矩形限:最常错,斜边区域内层限要带外层变量。
  • 内层限含内层变量:积完还剩 \(y\),说明限写反了。
  • 该取补不取补:\(X+Y>c\) 之类先看补集是否更简单(一块 vs 两块)。
  • pdf/cdf 混淆:给 cdf 却又去积分;给 pdf 求 cdf 忘了积分。
  • 反解常数忘检验非负:\(\iint f=1\) 定 \(k\) 后要保证 \(f\ge0\)。

📝 课本自测(做完对答案)

先自己做,再翻课本 Solutions 核对。

  • 24.1令 \(\iint f=1\) 反解常数 \(k\)(\(f=kxy^2\),三角支撑 \(0<x<y<1\))。
  • 24.2反解 \(k\):\(f=kx^{-3}e^{-y/3}\),\(1<x<\infty,\,1<y<\infty\)。
  • 24.3\(f=\tfrac34x\),\(0<x<2,\,0<y<2-x\),求 \(P[X>1]\)(斜边支撑)。
  • 24.5给联合 cdf \(F\),求 \(P[X>2]\) —— 用 cdf 作差,别积分。
  • 24.6[S01:5] 保险索赔 \(x+y<1\) 三角支撑,求区域概率。

🔎 60 秒快测(口算/setup)

  • 支撑 \(0<x<y<1\),要 \(\int\!\!\int f\,dy\,dx\):内层 \(y\) 的限是多少?外层 \(x\)?
  • \(f=1.2(x^2+y)\) 在单位方,\(P[X+Y>0.8]\) 该直接算还是取补?
  • 给联合 cdf \(F(x,y)\),想求密度 \(f(x,y)\),做什么运算?
✅参考答案:① 内层 \(y:\,x\to1\),外层 \(x:\,0\to1\)(内层随 \(x\));② 取补算 \(P[X+Y\le0.8]=0.14336\),得 \(0.85664\);③ 求混合二阶偏导 \(\partial^2F/\partial x\,\partial y\)。

收尾

公式速查卡 · 连续型联合分布

学完今天,这张卡随手能默写,就算过关了。

密度 / 概率

\(f(x,y)\ge0,\ \iint_{\mathbb{R}^2}f=1\)

\(P[(X,Y)\in R]=\iint_R f\,dA\)

矩形:内外层常数限

非矩形:内层限含外层变量;斜边优先取补

cdf ↔ pdf

\(F(x,y)=P[X\le x,Y\le y]=\int_{-\infty}^{x}\!\int_{-\infty}^{y}f\,dt\,ds\)

\(f(x,y)=\dfrac{\partial^2F}{\partial x\,\partial y}\)

\(F(\infty,\infty)=1,\ F_X(x)=F(x,\infty)\)

给 cdf → 代值作差;给 pdf → 积分

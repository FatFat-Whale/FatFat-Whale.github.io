第二十三章 · Order Statistics 次序统计量

从一组 iid 样本 \(X_1,\dots,X_n\) 里,把它们由小到大排序,取出最小、最大或第 \(k\) 个——这就是次序统计量。考试里最常出的是最大值(maxima)与最小值(minima)的分布。核心只有两招:先写 CDF(全都 \(\le x\) / 全都 \(>x\)),再求导得 pdf。对照课本 Lesson 23 精读。

Ch23 Order Statistics 导学已备 课本 p.409–413(理论 + Table 23.1) 前置 CDF/pdf · 独立性

★ 本章拿走 3 句话

  1. max 用 F 连乘:「最大 \(\le x\)」=「全体 \(\le x\)」→ \(F_{\max}=(F)^n\),密度 \(nF^{n-1}f\)。
  2. min 用 S 连乘:「最小 \(>x\)」=「全体 \(>x\)」→ \(S_{\min}=(S)^n\),CDF \(=1-(1-F)^n\)。
  3. 两条速用结论:iid 指数取最小 \(\sim\text{Exp}(\theta/n)\);\([0,\theta]\) 均匀第 \(k\) 个期望 \(=\dfrac{k\theta}{n+1}\)。

核心知识点及考题策略

先扫一遍公式卡热身(带「推」标的悬停看推导);学完回来对照「考点 + 套路」自检,这一节全拿下才算过关。

最大 / 最小

推最大:\(F_Y=(F_X)^n\),\(f_Y=n(F_X)^{n-1}f_X\)

推最小:\(F_Y=1-(1-F_X)^n\)

　　　\(f_Y=n(1-F_X)^{n-1}f_X\)

推指数最小 \(\sim\text{Exp}(\theta/n)\)

第 k 个 / 均匀

推\(f_{X_{(k)}}=\dfrac{n!}{(k-1)!\,1!\,(n-k)!}F^{k-1}f(1-F)^{n-k}\)

均匀 \([0,\theta]\):\(E[X_{(k)}]=\dfrac{k\theta}{n+1}\)

判别:max→全 \(\le x\);min→全 \(>x\)

🎯 本章核心考点

  • 写出最大值 \(X_{(n)}\):\(F_{\max}=[F]^n\)、pdf \(=n[F]^{n-1}f\);与最小值 \(X_{(1)}\):\(S_{\min}=[S]^n\)、\(F_{\min}=1-[1-F]^n\)。
  • 套第 \(k\) 个次序统计量的 pdf 公式(带多项式系数)。
  • 记住两条速用结论:\(n\) 个 iid 指数取最小 \(\sim\text{Exp}(\theta/n)\);\([0,\theta]\) 均匀第 \(k\) 个期望 \(=\dfrac{k\theta}{n+1}\)。

🧭 题型套路(SOA 真题)

  1. 关键词:the largest / smallest of the claims、the first of n machines to fail(第一台故障 = min)、the last to fail(= max)。
  2. 高频组合:\(n\) 台设备寿命 iid 指数,"第一次故障时间" \(\sim\text{Exp}(\theta/n)\) 一步出答案;各自均值不同时 hazard 相加,\(1/\theta_{\min}=\sum 1/\theta_i\)(如练 23.3)。
  3. 最大索赔期望三步:先 \(F_X\) → \(f_Y=nF^{n-1}f\) → \(\int x\,f_Y\,dx\),积分对象是密度不是 CDF。
  4. 头号陷阱:min 误用 \(F^n\)(把最小当最大),对比表刻脑。

步骤用的公式 / 定理出处
① 密度积分得单笔 CDF\(F(x)=\int_1^x 3t^{-4}dt=1-x^{-3},\ x>1\)L05
② 三笔最大值的 CDF\(G(y)=F(y)^3=(1-y^{-3})^3\)本章

① 写平移指数CDF\(F(y)=1-e^{-(y-5)}\)L20
② 最小次序统计量密度\(g_1(y)=3f(y)[1-F(y)]^2=3e^{-3(y-5)}\)本章

📖 三步的关系:备料 → 装配 → 认形
① 备料(密度→CDF):\(F(y)=\int_5^y e^{-(t-5)}dt=1-e^{-(y-5)}\)。这是「平移指数」——密度自带 \((y-5)\),变量定义从 5 起跳(分布本身带平移,不是换元)。①的目的只有一个:给②准备生存函数 \(S(y)=1-F(y)=e^{-(y-5)}\)。
② 装配(min 的密度公式):\(g_1(y)=n\,f(y)\,S(y)^{n-1}=3f(y)[1-F(y)]^2\)。公式的两种来路:(a) 生存函数法——独立三件都活过 y 连乘 \(P[\min>y]=S^3\),对 \(G_{\min}=1-S^3\) 求导链式出 \(3S^2f\);(b) 直接数——3 选 1 谁最先坏 × 它的密度 × 另两件仍活着 \(S^2\)。代入:\(3e^{-(y-5)}\cdot e^{-2(y-5)}=3e^{-3(y-5)}\)。①造 S、②用 S,与 23.1 的③④同款生产-消费。
③ 认形(免积分):\(3e^{-3(y-5)}\ (y>5)\) 正是「率 3 的指数整体右移 5」的密度 → \(Y_{(1)}=5+W,\ W\sim\)Exp(均值 \(\tfrac13\))。期望平移律 \(E(c+W)=c+E(W)\):\(E=5+\tfrac13=5.33\) (B)——不用 \(\int y\,g\,dy\) 硬积,认出分布直接抄均值。

先扫雷 · 必读

符号破译:下标是"谁",括号是"在哪问"

这一章的乱,九成来自符号:同一个 \(x\) 点上同时站着老变量 \(X\)(单个零件)和新变量 \(Y\)(最大/最小值),全靠下标认人。先把这张表钉死,再往下学。

破译表:每个符号只回答一个问题

写法读法意思
\(F_X(x)\)"X 的 CDF 在 x 点的值"单个零件寿命 \(\le x\) 的概率(0 → 1 的 S 形累积)
\(f_X(x)\)"X 的 PDF"单个零件寿命落在 \(x\) 附近的密度(= CDF 的斜率)
\(S_X(x)\)"X 的生存函数"\(=1-F_X(x)\),活过 \(x\) 的概率
\(F_Y(x)\)"Y 的 CDF 在 x 点的值"新造变量 \(Y\)(如最大值)\(\le x\) 的概率——和 \(F_X\) 是两条不同的曲线
\(f_Y(x)\)"Y 的 PDF"新变量的密度,永远由 \(F_Y\) 求导而来
🧠下标 = 谁的分布;括号 = 在数轴哪个点提问。括号里写 \(x\) 写 \(t\) 都无所谓,它只是横轴占位符;\(F_Y(x)\) 就是一句话:"\(Y\le x\) 吗?"

CDF 与 PDF 之间只有一条来回路,这一章全程只用它:\(F\xrightarrow{\;d/dx\;}f\)(求导),\(f\xrightarrow{\;\int_{-\infty}^{x}\;}F\)(积回去)。书上两套排序记号 \(Y_k\) 与 \(X_{(k)}\) 是同一个东西:\(X_{(1)}=Y_1=\) 最小,\(X_{(n)}=Y_n=\) 最大。

三层楼:分布 → 随机变量 → 观测值(样本到底是谁?)

层是什么灯泡例记号
分布抽奖机的规格说明书"同款灯泡"的出厂规格\(U(0,10)\)、\(F_X\)
随机变量还没开奖的号码每个灯泡的寿命(装上时未知)\(X_1,X_2,X_3\)
观测值开奖后的具体数字真烧坏后记下 3.2 / 7.8 / 5.1 年小写 \(x_1,x_2,x_3\)

一个灯泡 = 一个变量 \(X_i\);三个凑一起才叫"一组样本"(sample of size 3,集体名词)——P 卷里"样本"直接翻译成"n 个 iid 随机变量"。裸 \(X\)(不带下标)是偷懒的通用代号:同款同分布,谈单个寿命写谁都一样,\(F_X\) 是三个变量共用的同一张说明书,连乘 \([F_X]^3\) 三次用的是同一个 \(F\)。

\(M=\max(X_1,X_2,X_3)\)、\(W=\min(\cdots)\) 不是从"一个 X"衍生,而是三个变量合成的新变量——"样本的函数"教材叫统计量(statistic),次序统计量因此得名;合成时靠独立才能把"三个全 ≤ x"拆成连乘。考试全程活在"开奖前"的世界(算概率与期望),小写观测值到统计推断课才唱主角。

灯泡故事:一个例子串起全章

车间装 3 个同款灯泡,寿命独立同分布 \(X\sim U(0,10)\)(年)。单个灯泡三件套:\(F_X(x)=\tfrac{x}{10}\),\(S_X(x)=1-\tfrac{x}{10}\),\(f_X(x)=\tfrac1{10}\)。现在造两个新变量:\(M=\) 最后坏的时刻(max),\(W=\) 第一个坏的时刻(min)。

① max 走 CDF——"最后坏的 ≤ 6" 就是 "三个全都 ≤ 6",独立连乘 \(F_M(6)=[F_X(6)]^3=0.6^3=0.216,\qquad F_M(x)=[F_X(x)]^n\)
② min 走生存——"第一个坏的 > 6" 就是 "三个全都还活着" \(S_W(6)=[S_X(6)]^3=0.4^3=0.064\ \Rightarrow\ F_W(6)=1-0.064=0.936\)

直觉核对:人多,最弱的先倒——6 年内"几乎必有一个先坏"(0.936),但"三个全坏"还早(0.216)。

③ 要 PDF?对 CDF 求导,没有第二条路 \(f_M(x)=\dfrac{d}{dx}[F_X(x)]^3=3[F_X(x)]^2 f_X(x)=3\Big(\dfrac{x}{10}\Big)^2\dfrac1{10}\)

这个式子自己会讲故事:3 选 1 当"最后坏的" × 另外两个已坏 \((F^2)\) × 它正好在 \(x\) 坏 \((f)\)。

④ 期望收尾 + 均匀站位捷径互相验证 \(E[M]=\int_0^{10}x\cdot\dfrac{3x^2}{1000}dx=7.5\)
\(E[X_{(k)}]=a+(b-a)\dfrac{k}{n+1}\ \Rightarrow\ E[M]=10\cdot\dfrac34=7.5\ ✓,\quad E[W]=10\cdot\dfrac14=2.5\)

站位直觉:3 个人把 \((0,10)\) 切成 4 等份,min/中位/max 分别站在 2.5 / 5 / 7.5。

💡考场三板斧:问 max ≤ ? → \(F^n\) 连乘;问 min > ? → \(S^n\) 连乘(问 min ≤ ? 记得取补!);要期望/PDF → 先写 CDF 再求导,均匀直接用站位公式。指数加送:\(n\) 个 iid \(\text{Exp}(\theta)\) 的 min 仍是指数、均值 \(\theta/n\)。

最大值 \(X_{(n)}\) 的分布

把样本排序后有两套记号:有人用 \(Y_i\),有人用 \(X_{(i)}\);\(X_{(1)}\) 是最小、\(X_{(n)}\) 是最大。先从最好推的最大值开始:最大值 \(\le x\),等价于每一个 \(X_i\) 都 \(\le x\)。

关键一步:"最大 \(\le x\)" = "全体 \(\le x\)" 考频 ★★★

课本 p.409 · 式 (23.1)

最大值 \(\le x\) ⟺ 每一个样本都 \(\le x\);独立同分布就把 CDF 连乘 \(n\) 次。

★ 必背CDF of the maximum $$F_Y(x)=\Pr[Y\le x]=\prod_{i=1}^{n}\Pr[X_i\le x]=\big(F_X(x)\big)^n$$

设 \(Y=\max(X_1,\dots,X_n)\)。\(Y\le x\) 当且仅当 \(X_i\le x\) 对每一个 \(i\) 都成立,独立性让概率可以连乘。对 \(x\) 求导即得密度(链式法则):

pdf of the maximum · (23.2) $$f_Y(x)=n\,\big(F_X(x)\big)^{n-1}\,f_X(x)$$
展开 · pdf 为什么是 nF^(n-1)f

对 \(F_Y=(F_X)^n\) 用链式法则:\(f_Y=n(F_X)^{n-1}\cdot F_X'=n(F_X)^{n-1}f_X\)。最后那个 \(f_X\) 最容易漏。

最大值密度右移:取最大把质量推向右尾
\([F(x)]^n\le F(x)\):\(n\) 次方把 CDF 压低、爬升更晚
例 23A课本 p.409 · 均匀分布

从 \([0,1]\) 均匀分布抽 3 个样本,求样本最大值大于 0.7 的概率。

均匀 \([0,1]\):\(F_X(x)=x\)。最大值 CDF \(F_Y(x)=x^3\)。故 \(\Pr[Y>0.7]=1-\Pr[Y\le0.7]=1-0.7^3=1-0.343\)。

Pr[Y > 0.7] = 1 − 0.7³ = 0.657

对偶做法

最小值 \(X_{(1)}\) 的分布

最小值的对偶技巧:直接算 CDF 不好写,但"最小 \(>x\)" = "全体 \(>x\)"很好写。所以先算生存函数(survival function),再补成 CDF。

关键一步:走生存函数 \(S=1-F\) 考频 ★★★

课本 p.410 · 式 (23.3)–(23.4)

最小值 \(>x\) ⟺ 全体 \(>x\)——先写生存函数的 \(n\) 次方,再补成 CDF。

★ 必背Survival function of the minimum $$S_Y(x)=\Pr[Y>x]=\prod_{i=1}^{n}\Pr[X_i>x]=\big(S_X(x)\big)^n=\big(1-F_X(x)\big)^n$$
🗝️max 找 F,min 找 S,都连乘 — 「最大 ≤ x」=全体 ≤ x → \(F^n\);「最小 > x」=全体 > x → \((1-F)^n\)。

设 \(Y=\min(X_1,\dots,X_n)\)。补成 CDF、再求导得密度:

CDF & pdf of the minimum · (23.3)(23.4) $$F_Y(x)=1-\big(1-F_X(x)\big)^n,\qquad f_Y(x)=n\big(1-F_X(x)\big)^{n-1}f_X(x)$$

与最大值完全对称:最大用 \(F^n\)、最小用 \((1-F)^n\)。

展开 · 从 S 补回 CDF 与 pdf

\(F_Y=1-S_Y=1-(1-F_X)^n\);求导时链式法则里内层导数是 \(-f_X\),与前面的负号相消,得 \(f_Y=n(1-F_X)^{n-1}f_X\)。

最大值 \(X_{(n)}\)最小值 \(X_{(1)}\)
等价事件「max \(\le x\)」= 全体 \(\le x\)「min \(>x\)」= 全体 \(>x\)
连乘对象CDF:\(F_Y=(F_X)^n\)生存函数:\(S_Y=(S_X)^n=(1-F_X)^n\)
CDF\((F_X)^n\)\(1-(1-F_X)^n\)
pdf\(n(F_X)^{n-1}f_X\)\(n(1-F_X)^{n-1}f_X\)
密度偏向右尾(取最大变大)左尾(取最小变小)
速用结论均匀 \([0,\theta]\):\(E=\dfrac{n\theta}{n+1}\)iid 指数:\(\min\sim\text{Exp}(\theta/n)\)
最小值密度左移:取最小把质量推向左尾
速用结论:\(n\) 个 iid \(\text{Exp}(\theta)\) 取最小 \(\sim\text{Exp}(\theta/n)\)
⭐指数最小值(高频):若 \(X_i\stackrel{iid}{\sim}\text{Exp}\)、均值 \(\theta\),则 \(S_X(x)=e^{-x/\theta}\),\(S_Y(x)=(e^{-x/\theta})^n=e^{-x/(\theta/n)}\)——最小值仍是指数,均值缩成 \(\theta/n\)。更一般地,速率(hazard)可加:各自均值 \(\theta_i\) 时最小值均值 \(1\big/\sum_{i=1}^n 1/\theta_i\)。
⚠️最易错:最小值必须用生存函数 \(n\) 次方 \((1-F)^n\),不是 \(F^n\)。写反就把"最小"当成"最大"了。判别口诀:max→全体 \(\le x\)→\(F^n\);min→全体 \(>x\)→\((1-F)^n\)。

第 \(k\) 个次序统计量

最大、最小只是 \(k=n\)、\(k=1\) 的特例。一般的第 \(k\) 小(如中位数)有统一密度公式。SOA 大纲说"probably won't be tested",但结构值得懂。

第 \(k\) 个次序统计量的密度 考频 ★

课本 p.410 · 式 (23.5) · Table 23.1

第 \(k\) 小落在 \(x\):\(k-1\) 个更小、恰 1 个在 \(x\)、\(n-k\) 个更大——系数数的是分堆方法数。

pdf of the k-th order statistic $$f_{X_{(k)}}(x)=\frac{n!}{(k-1)!\,1!\,(n-k)!}\,\big(F_X(x)\big)^{k-1}\,f_X(x)\,\big(1-F_X(x)\big)^{n-k}$$

组合意义(多项式系数):要让第 \(k\) 小恰好落在 \(x\) 附近,需要 \(k-1\) 个样本小于 \(x\)(概率 \(F^{k-1}\))、恰 \(1\) 个在 \(x\)(密度 \(f\))、\(n-k\) 个大于 \(x\)(概率 \((1-F)^{n-k}\));系数 \(\dfrac{n!}{(k-1)!\,1!\,(n-k)!}\) 数的是"哪些样本分到哪三堆"的方法数。

🗝️指数取最小,均值除 n;均匀第 k,期望 \(k\theta/(n+1)\) — 两条速用结论,考场直接抄。
三堆划分 → 多项式系数 \(\dfrac{n!}{(k-1)!\,1!\,(n-k)!}\)

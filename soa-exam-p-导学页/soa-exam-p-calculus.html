01
Derivatives

求导速查

基本规则

公式名字考试落点
\((x^n)' = n x^{n-1}\)幂法则多项式密度求众数、Beta 型密度
\((e^{ax})' = a\,e^{ax}\)指数指数/泊松族无处不在;注意链出的 \(a\)
\((\ln x)' = 1/x\)对数由 \(F\) 反解分位数后校验;对数变换密度
\((uv)' = u'v + uv'\)乘积法则\(x e^{-x}\) 型密度求众数、MGF 二阶导
\(\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}\)商法则比值型 MGF(如几何/负二项)求导取矩
\(\big(f(g(x))\big)' = f'(g)\,g'(x)\)链式法则变换密度法 \(f_Y = f_X(g^{-1})\left|\frac{dg^{-1}}{dy}\right|\) 的那个"内导"
★ 必背本课程最常做的三次求导 $$F'(x) = f(x) \qquad S'(x) = -f(x) \qquad M^{(n)}(0) = E[X^n]$$
🗝️一句话:CDF 求导得密度,生存函数求导带负号,MGF 在 0 处求 n 阶导得 n 阶矩。这三条覆盖了全书 80% 的求导动作——分布函数法(L34)、生存函数(L07)、MGF 取矩(L32)。

求众数(mode)的标准流程

① 对密度 \(f(x)\) 求导(常先取 \(\ln f\) 再求导,乘积变加法);② 解 \(f'(x)=0\);③ 查端点——定义域边界可能才是最大值(L10 的坑)。

02
Integrals

积分速查

基本反导

公式考试落点
\(\displaystyle\int x^n\,dx = \frac{x^{n+1}}{n+1}\;(n\neq-1)\)多项式密度归一化、均匀/Beta 型期望
\(\displaystyle\int \frac{1}{x}\,dx = \ln|x|\)密度 \(\propto 1/x^k\) 的 Pareto 型;\(n=-1\) 特例别硬套幂法则
\(\displaystyle\int e^{ax}\,dx = \frac{e^{ax}}{a}\)指数族一切计算;除以 \(a\) 别忘
\(\displaystyle\int \ln x\,dx = x\ln x - x\)偶见于对数密度的期望
★ 必背Gamma 技巧(全书最重要积分) $$\int_0^{\infty} x^{n} e^{-x/\theta}\,dx = n!\,\theta^{\,n+1}$$
🗝️「n 阶乘 θ 的 n+1 次」——看到 \(\int_0^\infty x^n e^{-x/\theta}\) 直接写答案,不做分部积分。特例:\(\int_0^\infty e^{-x/\theta} = \theta\),\(\int_0^\infty x e^{-x/\theta} = \theta^2\),\(\int_0^\infty x^2 e^{-x/\theta} = 2\theta^3\)。速率版(\(e^{-ax}\)):\(\int_0^\infty x^n e^{-ax} dx = n!/a^{n+1}\)。

这一条就是指数分布全部矩的来源:\(E[X^n] = \frac{1}{\theta}\int_0^\infty x^n e^{-x/\theta} dx = n!\,\theta^n\)(L20);Gamma 分布归一化、MGF 推导(L32)同源。

期望的两种积分写法(L07 横竖两切)

定义式(竖切) vs 生存捷径(横切) $$E[X] = \int_0^{\infty} x\,f(x)\,dx \qquad\Longleftrightarrow\qquad E[X] = \int_0^{\infty} S(x)\,dx\ \ (X\ge0)$$

带免赔额/限额的赔付期望,横切版常少算一半的功夫:\(E[\min(X,u)] = \int_0^{u} S(x)dx\)、\(E[(X-d)_+] = \int_d^{\infty} S(x)dx\)。

03
Integration by Parts

分部积分与表格法

★ 必背Integration by parts $$\int u\,dv = uv - \int v\,du$$

表格法(tabular):\(x^n e^{ax}\) 型一次成型

左列对 \(x^n\) 连续求导到 0,右列对 \(e^{ax}\) 连续积分,斜线相乘、符号 \(+,-,+,\dots\) 交替相加。

符号求导列(u)积分列(dv)
\(+\)\(x^2\)\(e^{-x}\to -e^{-x}\)
\(-\)\(2x\)\(\to e^{-x}\)
\(+\)\(2\)\(\to -e^{-x}\)

得 \(\int x^2 e^{-x} dx = -(x^2+2x+2)e^{-x}+C\)。算一遍(定积分):代 \(0\to\infty\),\(\infty\) 端指数压死多项式取 0,\(x=0\) 端得 \(2\) → \(\int_0^\infty x^2 e^{-x} dx = 2 = 2!\),和 Gamma 技巧对上。

⚠️有限区间没有捷径:\(\int_0^{d} x e^{-x/\theta} dx\)(免赔额题常见)不能套 Gamma 技巧,老老实实表格法,或改用横切 \(\int S\) 绕开。
04
FTC & Leibniz

变限积分求导

★ 必背微积分基本定理 + 变限法则 $$\frac{d}{dx}\int_a^{x} f(t)\,dt = f(x) \qquad\qquad \frac{d}{dx}\int_{u(x)}^{v(x)} f(t)\,dt = f(v)\,v' - f(u)\,u'$$
动作哪里用
\(F(x)=\int_{-\infty}^x f\) 求导回 \(f\)分布函数法(L34):先算 \(F_Y\),再求导得 \(f_Y\)
上限是函数:\(\frac{d}{dy}\int_0^{g(y)} f = f(g)\,g'\)变换题 \(Y=g(X)\) 中 \(F_Y(y)=F_X(g^{-1}(y))\) 求导,链式自动冒出雅可比因子(L34)
对参数求导:\(\frac{d}{dd}\int_d^\infty (x-d)f\,dx = -\int_d^\infty f\,dx\)免赔额 \(d\) 的敏感度/反解 \(d\) 类题
05
Double Integrals

二重积分要点(联合分布主战场)

三步纪律

  1. 先画区域:支撑集 ∩ 事件区域,链式不等式(\(0<x<y<1\))就是边界线。
  2. 定内外次序:内层变量的上下限可以含外层变量,外层必须是常数。竖切 \(\int_x\) 外 \(\int_y\) 内,或横切反之——选让内层限"干净"的方向。
  3. 换序要换限:\(\int_0^1\!\!\int_x^1 \cdots dy\,dx = \int_0^1\!\!\int_0^y \cdots dx\,dy\)——限从图上重读,别照抄。
常用套件 $$P[(X,Y)\in A] = \iint_A f(x,y)\,dA \qquad f_X(x) = \int f(x,y)\,dy \qquad E[g] = \iint g\,f\,dA$$
🗝️联合均匀先算面积:密度为常数时一切概率 = 目标面积 ÷ 总面积(L25),几何法免积分。真要积分再动笔。
06
Series

级数速查(离散侧的"积分")

公式(\(|r|<1\))考试落点
\(\displaystyle\sum_{n=0}^{\infty} r^n = \frac{1}{1-r}\)几何分布归一化/尾概率(L18)、PGF 求值(L33)
\(\displaystyle\sum_{n=1}^{\infty} n r^{n-1} = \frac{1}{(1-r)^2}\)几何分布均值——上式逐项求导即得
\(\displaystyle\sum_{n=0}^{\infty} \frac{x^n}{n!} = e^{x}\)泊松归一化、泊松 MGF/PGF(L19/L32/L33)
\(\displaystyle\sum_{k=0}^{n} \binom{n}{k} a^k b^{n-k} = (a+b)^n\)二项归一化、二项 MGF \((1-p+pe^t)^n\)(L17/L32)
🗝️级数求导 = 离散版分部积分:想要 \(\sum n r^n\) 这类带 \(n\) 的和,对 \(\sum r^n\) 逐项求导再乘 \(r\)——几何/负二项的矩都这么来。

08
Exam FM

FM 的微积分落点(仅两处)

利息力(L01)与久期(L08)

Force of interest · 从余额函数到累积因子 $$\delta_t = \frac{A'(t)}{A(t)} = \frac{d}{dt}\ln A(t) \qquad\qquad AV = \exp\!\Big(\int_{t_1}^{t_2} \delta_t\,dt\Big)$$
Duration · 价格对利率的导数 $$D_{mod} = -\frac{P'(i)}{P(i)} \qquad\qquad \frac{\Delta P}{P} \approx -D_{mod}\,\Delta i + \frac{1}{2}C\,(\Delta i)^2$$

FM 只考"会积 \(\delta_t\)(多为多项式/\(\frac{a}{1+at}\) 型,后者积出 \(\ln\))"和"理解 \(P'(i)\)、\(P''(i)\) 是久期凸性的来源",不考复杂积分技巧。

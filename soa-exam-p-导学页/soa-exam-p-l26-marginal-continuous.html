01
Ch26 · Lesson 26 · 课本 p.447

边缘密度 — 把另一个变量积掉

离散时求边缘是"把另一个变量的概率加掉";连续时把求和换成积分——"把另一个变量积掉"。几何上就是沿那个方向把联合密度这座"山"压扁成一条曲线。

定义:连续型边缘密度 考频 ★★★

课本 p.447

把要丢的那个变量在支撑内的取值范围上积分,剩下的就是边缘密度。

★ 必背Marginal densities $$f_X(x)=\int_{-\infty}^{\infty} f(x,y)\,dy,\qquad f_Y(y)=\int_{-\infty}^{\infty} f(x,y)\,dx$$
🗝️要谁留谁,另一个积掉 — 求 \(f_X\) 就把 \(y\) 沿支撑内扫过的区间积掉,结果只准剩 \(x\)。

写 \(-\infty\) 到 \(\infty\) 只是形式;真正的上下限由支撑决定——积分只在 \(f>0\) 的区域进行。得到的 \(f_X\) 只是 \(x\) 的函数、\(f_Y\) 只是 \(y\) 的函数。

展开 · 离散 ↔ 连续对照,"边缘"名字的来历

离散版(L12):\(p_X(x)=\sum_y p(x,y)\)——把联合分布表的每一行加总,合计数写在表格页边(margin)上,"边缘分布"因此得名。连续版只是把"行求和"换成"沿 \(y\) 积分",其余原样平移。

沿 \(y\) 积掉 → 每个 \(x\) 处的"总高度" = \(f_X(x)\)
支撑 \(0\le x\le y\le1\):固定 \(x\),\(y\) 从 \(x\) 到 \(1\)(黄条)
Ex 26A 结果:两条边缘密度,各自积分 \(=1\)
💡直觉:边缘 = 把联合密度沿"要丢的方向"压扁投影。压扁前先看清"这条竖(或横)线在支撑里能扫多长",那段长度就是积分区间。
⚠️本章唯一大坑 — 积分限随变量变:支撑非矩形(三角、抛物、半圆…)时,\(y\) 的上下限是 \(x\) 的函数,不能当常数照搬。务必先画支撑、固定一个变量、读出另一个变量的扫描区间,再动笔积分。
02
课本例题 + 真题

例题:读支撑、定限、积分

流程永远三步:①画/读支撑 → ②固定要保留的变量,读另一变量的积分限 → ③积分,并用总积分为 1 复核。

例 26A课本 p.447 · 三角支撑

联合密度 \(f(x,y)=2\),支撑为三角 \(0\le x\le y\le 1\)(顶点 \((0,0),(1,1),(0,1)\))。求 \(X,Y\) 的边缘密度。

固定 \(x\):由 \(x\le y\le1\),\(y\) 从 \(x\) 积到 \(1\) —— \(f_X(x)=\displaystyle\int_{x}^{1}2\,dy=2(1-x)\),\(0\le x\le1\)。
固定 \(y\):由 \(0\le x\le y\),\(x\) 从 \(0\) 积到 \(y\) —— \(f_Y(y)=\displaystyle\int_{0}^{y}2\,dx=2y\),\(0\le y\le1\)。
复核:\(\int_0^1 2(1-x)\,dx=1\),\(\int_0^1 2y\,dy=1\)。✔

f_X(x)=2(1−x)　f_Y(y)=2y
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① 边缘密度定义:把另一变量积掉\(f_X(x)=\int f(x,y)\,dy,\ \ f_Y(y)=\int f(x,y)\,dx\)本章 §01
② 读三角支撑定限(限随变量变)\(f_X(x)=\int_x^1 2\,dy=2(1-x),\ \ f_Y(y)=\int_0^y 2\,dx=2y\)本章 §01
③ 复核:密度总积分 = 1\(\int_0^1 2(1-x)\,dx=1,\ \ \int_0^1 2y\,dy=1\)L05 密度性质 ↗
真题 26.8[110-S88:4] · 半圆支撑

\((X,Y)\) 在上半单位圆盘 \(\{x^2+y^2\le1,\ x>0\}\) 上均匀分布。求 \(X\) 的边缘密度,\(0\le x\le1\)。

半圆盘面积 \(=\pi/2\),故联合密度 \(f=\dfrac{2}{\pi}\)(均匀)。固定 \(x\),\(y\) 从 \(-\sqrt{1-x^2}\) 到 \(\sqrt{1-x^2}\):
\(f_X(x)=\displaystyle\int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}}\frac{2}{\pi}\,dy=\frac{2}{\pi}\cdot 2\sqrt{1-x^2}=\frac{4}{\pi}\sqrt{1-x^2}\)。

(A) f_X(x) = (4/π)·√(1−x²)
🧩 步骤 ↔ 公式对照
步骤用的公式 / 定理出处
① 均匀 → 联合密度 = 1/支撑面积\(f(x,y)=\tfrac{1}{\pi/2}=\tfrac{2}{\pi}\)L25 联合均匀 ↗
② 固定 \(x\) 读 \(y\) 的扫描区间(限随 \(x\) 变)\(-\sqrt{1-x^2}\le y\le \sqrt{1-x^2}\)本章 §01
③ 积掉 \(y\) 得边缘密度\(f_X(x)=\int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}}\tfrac{2}{\pi}\,dy=\tfrac{4}{\pi}\sqrt{1-x^2}\)本章 §01
📝看点:26.8 的积分限 \(\pm\sqrt{1-x^2}\) 是彻底的 \(x\) 的函数,把"限随变量变"演到极致;均匀密度 = \(1/\text{面积}\)。
03
顺带一条重要判据

独立性:因式分解 + 矩形支撑

求完边缘,常接着问 \(X,Y\) 是否独立。连续情形的充要条件有两个必须同时满足的部分,漏了支撑那条最容易错。

连续型独立的充要条件 考频 ★★

课本 p.447

独立 = 可因式分解 + 矩形支撑,两条缺一不可。

★ 必背Independence (continuous) $$f(x,y)=g(x)\,h(y)\quad\text{且支撑为矩形(可含}\ \pm\infty\text{)}$$
🗝️可拆还得方 — 密度拆成 \(g(x)h(y)\) 只是半张票,支撑还必须是矩形才算真独立。

缺一不可:①联合密度能拆成"只含 \(x\) 的因子 × 只含 \(y\) 的因子";②支撑是矩形——即 \(x\) 的范围不依赖 \(y\)、\(y\) 的范围也不依赖 \(x\)。

例 26A 里 \(f=2\)(可拆),但支撑是三角不是矩形(\(y\ge x\) 把两变量捆在一起),故 \(X,Y\) 不独立。若独立,则 \(f(x,y)=f_X(x)f_Y(y)\)。

可拆 + 矩形支撑可拆 + 非矩形支撑
例子\(f=6e^{-2x-3y}\),\(x>0,\ y>0\)例 26A:\(f=2\),\(0\le x\le y\le1\)
独立?独立,且 \(f=f_X\,f_Y\)一定不独立
原因两变量取值范围互不牵制\(y\ge x\) 把两变量绑在一起
⚠️最爱设的陷阱:密度看着能因式分解就判独立——错。只要支撑上出现 \(0<x<y\)、\(x^2\le y\)、\(x^2+y^2\le1\) 这类把两变量绑在一起的约束,支撑非矩形,一定不独立。

直觉 · 拓展

换个角度看懂它

💡 直觉例子:天际线与页边合计

把联合密度想成一座密度山(城市人口密度图)。求 \(f_X\) 就是从南往北把整座城压扁:每个 \(x\) 处,把这条南北向街道上的"人口"全部叠起来——压扁后的轮廓线就是 \(X\) 的边缘密度,山越宽 / 越高的地方轮廓越高。

俯视之城:26.8 的半圆城,密度处处 \(2/\pi\),\(x\) 处南北街长 \(2\sqrt{1-x^2}\)
天际线:\(f_X(x)=\tfrac{4}{\pi}\sqrt{1-x^2}\) — 密度均匀时街越长,轮廓越高

"边缘"这个名字来自离散版:在联合分布表的页边(margin)写下每行、每列的合计,就是边缘分布;连续版只是把"行合计"换成"沿 \(y\) 积分"。压扁之前先看清"这条街在城里有多长"——那就是积分限。

🎯 SOA 题型套路

  1. 积分限画图先行:真题支撑几乎全是三角 / 梯形 / 抛物(\(0<x<y<2\)、\(x^2\le y\le x\)、半圆盘),先画支撑、固定保留变量、读出另一变量的扫描区间。
  2. 三重自检:结果只含保留变量;附上定义域;\(\int f_X=1\)。
  3. 均匀支撑先算面积:\(f=1/\text{面积}\)(如 26.8 半圆盘 \(f=2/\pi\))。
  4. 判独立的题:先看支撑是不是矩形,再看可不可拆——顺序反了就掉坑。

巩固

易错点 + 自测

⚠️ 高频易错点

  • 积分限当常数照搬:非矩形支撑上 \(y\) 的限是 \(x\) 的函数,先画图再定限。
  • 积错变量:求 \(f_X\) 就要把 \(y\) 积掉(结果只剩 \(x\)),别把 \(x\) 积掉了。
  • 忘写定义域:边缘密度必须附上 \(x\)(或 \(y\))的有效区间。
  • 因式分解 = 独立:漏看支撑是否矩形,是最常见失分点。

📝 课本自测(做完对答案)

先自己做,再翻课本 Solutions 核对。

  • 26.2 [110-S83:3]\(f=\tfrac{5}{16}xy^2,\ 0<x<y<2\),求 \(f_X\)。
  • 26.3 [110-S85:14]\(f=4x,\ 0<x<\sqrt{y}<1\),求 \(f_Y\)。
  • 26.4 [110-S90:14]\(f=\tfrac{12}{25}(x+y^2),\ 1<x<y<2\),求 \(f_Y\)。
  • 26.7 [S03:10]\(f=15y,\ x^2\le y\le x\),求 \(f_Y\)。

🔎 60 秒快测(先定限,再心算)

  • \(f=x+y,\ 0<x<1,\ 0<y<1\)(矩形):求 \(f_X(x)\)。
  • \(f=4x,\ 0<x<\sqrt{y}<1\):固定 \(y\),\(x\) 的积分限是?求 \(f_Y(y)\)。
  • 上题的 \(X,Y\) 独立吗?为什么?
✅参考答案:① \(f_X=\int_0^1(x+y)\,dy=x+\tfrac12\)(26.5 答案 E);② \(x\) 从 \(0\) 到 \(\sqrt{y}\),\(f_Y=\int_0^{\sqrt y}4x\,dx=2y\)(26.3 答案 B);③ 不独立——支撑 \(x<\sqrt y\) 非矩形,\(x\) 的上限依赖 \(y\)。

公式速查卡 · 连续型边缘分布

边缘密度

\(f_X(x)=\displaystyle\int f(x,y)\,dy\)(积掉 \(y\))

\(f_Y(y)=\displaystyle\int f(x,y)\,dx\)(积掉 \(x\))

积分限由支撑决定,非矩形时随变量变

复核:\(\int f_X\,dx=1\)

独立判据

独立 ⟺ \(f(x,y)=g(x)h(y)\)

且支撑为矩形

独立时 \(f=f_X(x)\,f_Y(y)\)

支撑绑住两变量(三角/抛物/圆)⟹ 不独立

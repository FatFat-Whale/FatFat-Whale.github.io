Exam P · 习题串讲 E01 · one scenario, eight lessons · L07–L13 + L16

你是精算师:一份骑手意外险,串起 L07–L13 + L16

立一个场景,把八章的概念全部放进真实要回答的问题里:先定价(均值),再问稳不稳(方差),再问要备多少钱(分位数),两个险种一起赔时风险怎么叠(联合分布 → 协方差),最后按骑手分组重新算一遍账(双重期望)。所有数字手算可验,前后互相呼应。

💡 每题右上角的概念标签可点:弹窗直接显示对应知识卡片(定义 / 白话 / 公式 / 算一遍);卡片里的虚线词还能继续点,链式跳卡。

配套章节: L07 均值 · L08 方差与矩 · L09 分位数 · L10 众数 · L11 联合分布 · L12 联合矩 · L13 协方差 · L16 双重期望(前置 L14 条件矩)

底座产品与数据

你在保险公司做产品:外卖骑手意外险,按单收保费。历史数据显示,每一单的理赔额 X(千元)只有三种结局:

理赔额 x(千元)0(没出险)1(小磕碰)4(大事故)
概率 p(x)0.50.40.1

整个第一幕只用这一张表。第二幕另起一套二维数据,把赔付拆成"车损 + 医疗"两块看联合行为(数字与第一幕的 X 不衔接,第三幕才回来对上 Q1/Q3)。

第一幕 · 单变量(L07 – L10)

只看一个数:这张保单该收多少、风险多大

Q1保费底线定多少?

老板问:每单至少收多少钱,长期才不亏本?

求平均每单赔出去多少,即 E[X]。

  1. 期望 = 每种结局 × 各自概率,加总:
E[X] = 0(0.5) + 1(0.4) + 4(0.1) = 0.8 千元
  1. 再加运营成本:每单固定成本 0.1 千 + 理赔额的 10% 手续费,总成本 C = 0.1 + 1.1X。期望的线性直接搬系数(不需要任何独立性、也不用重算分布):E[C] = 0.1 + 1.1(0.8) = 0.98 千。

概念落位:期望 = 概率加权平均,是"长期平均账"。定价、成本核算全从它出发;线性 E[aX+b] = aE[X]+b 让你不必重新求分布。

Q2推出"限额 2 千"的便宜版,该收多少?

市场部想推便宜版:单次最多赔 2 千,超出不管。新产品的纯保费?

求 E[min(X, 2)]。

  1. LOTUS:不用求 min(X,2) 的分布,直接"逐个结局套函数再加权"——min 值分别是 0、1、2:
E[min(X,2)] = 0(0.5) + 1(0.4) + 2(0.1) = 0.6 千元
  1. 生存函数捷径验算(整数取值时 E[X] = Σ P[X>k]):P[X>0] = 0.5,P[X>1] = 0.1,截到 2 为止 → 0.5 + 0.1 = 0.6 ✓。顺手看全额版:0.5 + 0.1 + 0.1 + 0.1 = 0.8 ✓ 对上 Q1。

概念落位:LOTUS——"求 g(X) 的期望不必先求 g(X) 的分布";生存函数捷径把期望变成"把尾巴概率一层层摞起来",限额/免赔额题(SOA 最爱)全是它。

Q3两款产品同样均价,哪款更"险"?

公司另有一款手机碎屏险:每单要么赔 0.5 千(50%)、要么赔 1.1 千(50%)。均值同样是 0.8 千。财务问:同样收 0.8 的纯保费,哪条业务线波动大、更吃资本金?

均值分不出高下,比方差 / 标准差。

  1. 骑手险:先用 LOTUS 求二阶原点矩 E[X²] = 0(0.5) + 1(0.4) + 16(0.1) = 2.0,再用捷径公式:
Var(X) = E[X²] − (E[X])² = 2.0 − 0.64 = 1.36,  σ_X ≈ 1.17 千
  1. 碎屏险只有两个值——两点捷径 Var = (b−a)²p(1−p) = (1.1−0.5)²(0.5)(0.5) = 0.09,σ = 0.3 千。
  2. 跨业务线比,用无量纲的变异系数 CV = σ/μ:骑手险 1.17/0.8 ≈ 1.46,碎屏险 0.3/0.8 ≈ 0.38——每收 1 元保费,骑手险扛的波动接近 4 倍。

概念落位:均值回答"平均赔多少",方差/标准差回答"围着平均晃多凶"。方差是面积量(千元²),σ 开根回到千元才能跟保费比;CV 再除掉均值,不同规模的业务才可同台比较。

Q4要备多少准备金,才能扛住 9 成的单子?

监管要求:证明单笔准备金能覆盖 90% 的理赔场景。备多少?

求 90 分位数 π₀.₉ = min{x : F(x) ≥ 0.9}。

  1. 先摞出 CDF:F(0) = 0.5,F(1) = 0.9,F(4) = 1。
  2. 找 F 首次爬到 ≥ 0.9 的位置:F(1) = 0.9,正好够 → π₀.₉ = 1 千元。
  3. 若监管改口要 95%:F(1) = 0.9 < 0.95,得跳到下一个值 → π₀.₉₅ = 4 千元。离散分布的分位数会这样"跳崖",所以定义用 ≥ 而不是 =。
  4. 顺带:中位数 = min{x : F(x) ≥ 0.5} = 0——一半以上的单子压根不赔钱。

概念落位:期望管"平均",分位数管"最坏的那批"。准备金、风控红线(VaR 就是分位数)看的都是位置量,不是均值。注意 90→95 一步,准备金翻 4 倍——尾巴贵。

Q5客服该按哪种结局准备?为什么"平均 0.8"没人赔到?

客服主管:"你们说平均赔 0.8 千,可我天天处理的都是不赔的单子?"

最常见的结局是什么?三个"中心"为何分家?

  1. 众数 = 概率最大的取值 = argmax p(x) = 0(0.5 最高)——客服的日常确实是"不赔"。
  2. 三中心排队:众数 0 = 中位数 0 < 均值 0.8。均值被 10% 的大事故拽上去了。
  3. 拽多狠?算偏度:E[(X−0.8)³] = 0.5(−0.8)³ + 0.4(0.2)³ + 0.1(3.2)³ = 3.024,除以 σ³ ≈ 1.59,偏度 ≈ +1.9——明显右偏(正偏)。

概念落位:均值、中位数、众数是三种"中心",只有对称分布才重合。保险理赔、收入、房价这类右偏数据,"平均数"永远高于"大多数"——偏度就是量化这个分家程度的三阶矩。

第二幕 · 双变量(L11 – L13)

理赔拆成两块:车损 + 医疗,一起赔时风险怎么叠

底座 2联合分布表

精算部把每单理赔拆成车损 C(0 或 2 千)和医疗 M(0 或 1 千)。一场大事故往往两头都赔——历史数据的联合 pmf:

p(c, m)M = 0M = 1边缘 p(c)
C = 00.500.100.60
C = 20.100.300.40
边缘 p(m)0.600.401.00

四个格子加总 = 1。这张 2×2 小表接下来回答第二幕所有问题。

Q6只看车损业务线,分布长什么样?

车损理赔外包给修理厂结算,对方只关心 C 自己的分布。

从联合表提取 C 的边缘分布。

  1. "求谁的边缘,消掉另一个"——按行加总:P[C=0] = 0.5 + 0.1 = 0.6,P[C=2] = 0.1 + 0.3 = 0.4(表右侧绿列)。
  2. 由此 E[C] = 2(0.4) = 0.8,E[M] = 1(0.4) = 0.4(列加总同理)。
  3. 两点捷径顺手出方差:Var(C) = (2−0)²(0.6)(0.4) = 0.96;Var(M) = 1²(0.6)(0.4) = 0.24。

概念落位:边缘分布 = 站在一个变量的立场把另一个"扫掉"(离散加总,连续积分)。它丢掉的正是"两者如何联动"的信息——所以接下来的问题边缘表答不了。

Q7车损和医疗独立吗?能不能各算各的?

实习生提议:"两个险种分开建模,合并时直接相乘,多省事。"

检验 p(c,m) = p(c)·p(m) 是否格格成立。

  1. 试左上角一格:p(0,0) = 0.50,而 p(0)·p(0) = 0.6 × 0.6 = 0.36。0.50 ≠ 0.36,一格不合,独立性一票否决。
  2. 方向也能读出来:0.50 > 0.36,"两头都没事"比独立假设更常见;同理 p(2,1) = 0.30 > 0.4×0.4 = 0.16,"两头都出事"也偏多——同生共死,正向联动。
  3. 乘积期望侧面再证:E[CM] = 2×1×0.30 = 0.6(只有右下格非零),而 E[C]E[M] = 0.8×0.4 = 0.32。0.6 ≠ 0.32 → 不独立。(独立 ⇒ E[CM]=E[C]E[M];注意箭头方向,逆命题不成立,见 Q11。)

概念落位:独立性判定 = 因式分解逐格验 + 支撑必须是矩形(本例 2×2 满格,矩形没问题,栽在数值上)。"加法自由、乘法要独立":E[C+M] 谁都能拆,E[CM] 只有独立才能拆。

Q8每单总赔付平均多少?——这步不需要独立

财务只要一个数:这套双险种保单每单总赔付 S = C + M 平均多少。

求 E[S] = E[C + M]。

  1. 线性直接拆(刚证了不独立也没关系,线性恒成立):E[S] = E[C] + E[M] = 0.8 + 0.4 = 1.2 千。
  2. 不放心?用联合矩万能公式"值 × 概率"双和验算:S 的四种结局 0、1、2、3,E[S] = 0(0.5) + 1(0.1) + 2(0.1) + 3(0.3) = 1.2 ✓。

概念落位:E[g(C,M)] = ΣΣ g·p 是二维版 LOTUS,万能但笨;期望的线性是快刀,且对任意相关结构都成立——这是它和方差最大的区别,下一问立刻对比。

Q9总赔付的风险呢?独立假设会闯什么祸?

资本金按总赔付 S 的波动配置。实习生已经按"独立"把两条线的方差直接加了。

求真实的 Var(S),和独立假设差多少。

  1. 先算协方差(用 Q7 的现成料):Cov(C,M) = E[CM] − E[C]E[M] = 0.6 − 0.32 = +0.28。正号 = 同向联动,和 Q7 读出的"同生共死"一致。
  2. 和的方差,交叉项登场:
Var(S) = Var(C) + Var(M) + 2Cov = 0.96 + 0.24 + 0.56 = 1.76
  1. 实习生的独立版:0.96 + 0.24 = 1.20,把风险低估了 32%(σ:1.33 → 误报 1.10)。资本金照 1.20 配,大灾年直接穿仓。
  2. 直接验算不含糊:E[S²] = 0(0.5) + 1(0.1) + 4(0.1) + 9(0.3) = 3.2,Var(S) = 3.2 − 1.2² = 1.76 ✓。

高频错:漏交叉项 2Cov。期望的加法免费,方差的加法要付"相关税"——正相关业务打包,风险叠浪(+2Cov);负相关才互相对冲(−)。Var(C−M) 也是两方差相加、只有协方差项变号。

Q10这个联动到底多强?换成"元"记账会变吗?

风控委员会看不懂 "Cov = 0.28 千元²" ——单位古怪,又听说换记账单位数字会变。

给一个无量纲、换单位不变的联动强度。

  1. σ_C = √0.96 ≈ 0.98,σ_M = √0.24 ≈ 0.49,乘积恰好 √(0.96×0.24) = √0.2304 = 0.48。
ρ = Cov / (σ_C · σ_M) = 0.28 / 0.48 ≈ 0.58
  1. 中等偏强正相关。换算成"元"验证:Cov 变 0.28 × 10⁶(双线性,两边各放大 1000 倍),σ 各放大 1000 倍,分母同步 × 10⁶——ρ 纹丝不动。这就是"协方差双线性 Cov(aC, bM) = ab·Cov"的用处。
  2. ρ 的天花板:ρ = ±1 当且仅当 M 是 C 的严格线性函数。0.58 离 1 还远——联动强,但没到"知一个就知另一个"。

概念落位:协方差给方向(±),数值被单位污染;除以两个 σ(各自的"一格波动")后变成纯数 ρ ∈ [−1,1],跨险种、跨单位可比。加常数(如每单固定补贴)两者都不动——联动只关乎"偏离各自均值"的部分。

Q11尾声:ρ = 0 能收工吗?

另一款产品测出 Cov = 0,实习生宣布"两险种独立,分开建模!"

不相关 ⇒ 独立吗?

  1. 不行,箭头是单向的:独立 ⇒ Cov = 0;反过来推不动。
  2. 标准反例:X 取 −1, 0, 1 各 ⅓,令 Y = X²。E[XY] = E[X³] = 0 = E[X]E[Y] → Cov = 0;但 Y 完全由 X 决定,独立无从谈起。
  3. 原因:协方差只探测线性联动。Y = X² 是纯弯曲关系——左右对称,带符号矩形正负恰好抵消,协方差瞎了。

高频陷阱:ρ = 0 只说明"没有线性关系",非线性依赖照样存在。唯一例外是二元正态(L31):那里 ρ = 0 才 ⟺ 独立。

第三幕 · 分组视角(L16,前置 L14)

换个切法:按骑手分组,把前两幕的账重算一遍

底座 3分组报表

运营部把骑手分成两类,只给你分组汇总报表(拿不到逐单明细):

组 G占比组内均赔 E[X | G](千元)组内方差 Var(X | G)
老手60%0.40.8
新手40%1.41.6

条件期望 E[X | G] 在这里是个随机变量:抽到哪个骑手、落进哪组,它才取定 0.4 或 1.4——这是 L14 → L16 的关键一跃。

Q12只有分组报表,整体均赔还能算吗?

老板又来对账:"Q1 说平均赔 0.8,可运营只给我分组数,你验一下。"

用全期望公式(塔式)从分组数恢复整体均值。

  1. 塔式:先组内平均,再按组权重平均:
E[X] = E[ E[X|G] ] = 0.6 × 0.4 + 0.4 × 1.4 = 0.24 + 0.56 = 0.8 千 ✓
  1. 正好对上 Q1 逐单算的 0.8——切块方式不影响总账,这就是塔式的本事:明细拿不到,分组够了。

概念落位:E[X|G=老手] = 0.4 是数;E[X|G] 是随机变量(取 0.4 或 1.4,概率 0.6/0.4)。全期望 = 对这个随机变量再求一次期望。SOA 最爱的出法:给分组条件均值 + 组权重,问总体。

Q13总波动 1.36,多少怪"组内随机",多少怪"组间差异"?

风控盯上 Q3 算出的 Var(X) = 1.36:"如果按组差异化定价,能消掉多少波动?"

用全方差公式(EVVE)把 1.36 拆成组内 + 组间。

  1. 组内项(平均的组内波动):E[Var(X|G)] = 0.6 × 0.8 + 0.4 × 1.6 = 0.48 + 0.64 = 1.12。
  2. 组间项(组均值本身的波动,两点捷径):Var(E[X|G]) = (1.4 − 0.4)² × 0.6 × 0.4 = 0.24。
Var(X) = E[Var(X|G)] + Var(E[X|G]) = 1.12 + 0.24 = 1.36 ✓ 对上 Q3
  1. 回答风控:分层定价最多消掉组间的 0.24(18%)——把两组各自收平各自的价,残余波动仍有 1.12。大头是组内随机,分组帮助有限;反之若组间占大头,分层定价才真值钱。

概念落位:EVVE = "组内平均方差 + 组均值的方差",全方差恒等式,一分不多一分不少(1.12 + 0.24 = 1.36 严丝合缝)。混合分布求方差、N 个随机个体求和(复合分布,L19 泊松复合往这儿走)全靠它;高频错:只算组间忘组内,或反之。

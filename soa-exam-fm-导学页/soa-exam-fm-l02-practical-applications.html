价值方程

所有款项折到同一比较日:\(\sum\text{流入}\cdot v^{s} = \sum\text{流出}\cdot v^{s}\)

比较日任选,解不变

等效利率

推\((1+j)^m=(1+i)^k\)

季度对半年 3%:\(j=1.03^{1/2}-1\);两年对季度 2%:\(1+j=1.02^8\)

未知时间

\(a^t=b\Rightarrow t=\ln b/\ln a\)

TVM:I/Y·PV·FV(反号)·CPT N

Rule of 72:\(t_{\times2}\approx 72/i\%\)(准:\(69.3/i\%+0.35\))

等时法

\(\bar t=\dfrac{\sum A_k t_k}{\sum A_k}\)

与利率无关 · \(i>0\) 时恒大于精确 t

未知利率

换元 \(x=1+i\) 或 \(x=v^n\) → 二次方程 → 舍平凡根

两条级数(Ex 2.9)

推\(d+d^2+\cdots=\frac{d}{1-d}=i\)

\(d+\frac{d^2}{2}+\frac{d^3}{3}+\cdots=-\ln(1-d)=\delta\)

🎯 本章核心考点

  • 任何现金流题:按规范画时间图(存取两侧、期单位、利率箭头、比较日竖箭),列价值方程一步解出。
  • 付款期与计息期不一致时,两条路线(等效利率 j / 原利率 i)都能走通,并默认选付款期路线。
  • 未知时间:LN 键 / TVM 键双解;心算翻倍时间(Rule of 72);等时法近似并说出"恒偏大"。
  • 未知利率:换元 \(x=1+i\) 或 \(x=v^n\) 化多项式,取合理根、舍平凡根。

§2a · 课本 p.73–77

价值方程与时间图

等价原则是整个利息理论的游戏规则:今天的 $1,000 与一年后的 $1,050(5% 下)对你无差异。接受它,所有问题都变成"把钱搬到同一天再比"。

价值方程 Equation of Value 考频 ★★★

课本 p.73

把所有款项折算到同一个比较日(comparison date),流入 = 流出,就是价值方程。

★ 必背Equation of Value $$\sum \text{流入}_k\,(1+i)^{\pm s_k} \;=\; \sum \text{流出}_k\,(1+i)^{\pm s_k}\qquad(\text{全体折到比较日})$$
🗝️先搬钱,再列式 — 不同日期的钱不能直接加减,必须先搬到同一张"桌子"(比较日)上。

两个必须注意的点(p.73):① 评估任何两笔钱必须用同一日期;② 任何日期都可当比较日——哪怕取 122 年后,两边同除 \(1.06^{120}\) 就回到原方程,解不变。

展开:引例(p.73)

两年后凑 $5,000:今天存 X、一年后再存 X,6% 有效。取时间 2 为比较日:\(X(1.06^2+1.06)=5{,}000\Rightarrow X=\$2{,}289.80\)。

付款期 vs 计息期:两条路线 考频 ★★★

课本 p.75–77 · Ex 2.3 / 2.4

付款频率与计息频率不一致时,要么换利率就付款期,要么换刻度就计息期——两种都必须会,默认走付款期。

★ 必背等效利率 Equivalent Rate $$(1+j)^{m}=(1+i)^{k}\qquad\text{如 } j=1.03^{1/2}-1\ (\text{季度}),\quad 1+j=1.02^{8}\ (\text{两年})$$
🗝️刻度跟付款走,利率跟刻度走 — 教材原话:利息函数在"计息期=付款期"时形式最简,等效利率不是整数无妨,计算器不在乎。
路线① 按付款期(推荐)路线② 按计息期
时间轴刻度一格 = 一个付款期(Ex 2.3:¼ 年,0–40)一格 = 一个计息期(Ex 2.3:½ 年,0–20)
用的利率等效利率 \(j=1.03^{1/2}-1=1.48892\%\)原有效利率 3%(付款落在 ½, 1, 1½ …)
何时用数值答案(绝大多数考题)答案留符号形式时(出题人用原利率符号)
展开:Ex 2.4(付款间隔超过 1 年也同样处理)

1994–2014 偶数年 1/1 各存 $500,名义 8% 季度复利,求末笔存款日 AV。付款期 = 2 年 = 8 个季度:\(1+j=1.02^8\Rightarrow j=17.1659\%\),\(AV=500[1+(1+j)+\cdots+(1+j)^{10}]\)。计息期法:¼ 年轴 0–80,存款每隔 8 格。时间刻度编号随意(1–11、8–88 都行),只要付款间距正确。

未知时间:对数、TVM 与等时法

价值方程里解 t——精确解靠对数或计算器 TVM 键;心算近似靠 Rule of 72;多笔付款合并成单笔靠等时法。

解未知时间 Unknown Time 考频 ★★

课本 p.77 · Ex 2.5–2.7

指数方程 \(a^t=b\) 两条路:LN 键取对数,或 TVM 键直接 CPT N。

对数解 $$500(1.04^t)=1{,}500\ \Rightarrow\ t=\frac{\ln 3}{\ln 1.04}=28.01\ \text{年}$$
🖩BA II Plus(Ex 2.5,p.77):4 I/Y 500 PV 1500 +/- FV CPT N → 28.01。PV 与 FV 必须反号,否则报错。
Rule of 72(翻倍时间心算) $$t_{\times 2}\approx\frac{72}{i\%}\qquad\text{更准: } t\approx\frac{69.3}{i\%}+0.35\quad(\ln 2\approx 0.693)$$
🗝️翻倍七十二 — 5% 下 72/5=14.4 年;精确值 14.2067,改良式 69.3/5+0.35=14.21 已准到小数两位。
展开:两笔付款在 t 与 2t(Ex 2.7 套路)

\(1{,}000(v^t+v^{2t})=1{,}200\)(4%):令 \(x=v^t\) 化二次 \(x^2+x-1.2=0\),取正根 \(x=.704159\),再 LN/TVM 反解 \(t=8.943\)。换元 → 二次 → 取正根 → 反解是本题型标准四步。

等时法 Method of Equated Time 考频 ★★

课本 p.78–79 · Ex 2.9 / 2.10

"多笔付款合并成一笔,该付在哪天?"——精确解要解方程;等时法用金额加权平均时间一步近似。

★ 必背Equated Time $$\bar t=\frac{\sum_k A_k\,t_k}{\sum_k A_k}\qquad\text{精确: }\Big(\sum_k A_k\Big)v^{t}=\sum_k A_k v^{t_k}$$
🗝️金额当权重,答案总偏晚 — \(\bar t\) 与利率无关,且正利率下恒大于精确 t(书 79:approximation always exceeds the exact answer)。

例(p.79):付款 5、1、15 于时点 1、3、10 → \(\bar t=\frac{5(1)+1(3)+15(10)}{21}=7.52\);i=5% 时精确 t=7.12。权重必须是金额——"不看金额平均时点"不是好近似。

例 2.10课本 p.79–80

Hannah 应收 100/200/300 于年末 4/10/12;改为在等时法时点一次收 600;6% 有效。两种安排的现值差 X?

\(\bar t=\frac{4(100)+10(200)+12(300)}{600}=10\);单笔 \(PV=600v^{10}=335.04\);原计划 \(PV=100v^4+200v^{10}+300v^{12}=339.98\)。单笔付在偏晚时点 ⇒ PV 偏小。

X = 4.94

① 等时法定合并时点:金额加权平均\(\bar t=\frac{\sum A_k t_k}{\sum A_k}=\frac{4(100)+10(200)+12(300)}{600}=10\)本章
② 单笔方案 PV\(600v^{10}=335.04\)L01 折现 ↗
③ 原计划 PV:逐笔折现求和\(100v^4+200v^{10}+300v^{12}=339.98\)本章

展开:Ex 2.9 的两个经典级数(选择题速杀)

无穷付款 \(d, \frac{d^2}{2}, \frac{d^3}{3},\dots\) 求 \(\bar t\):分子 \(d+d^2+\cdots=\frac{d}{1-d}=i\);分母 \(d+\frac{d^2}{2}+\cdots=-\ln(1-d)=\delta\)。答案 \(\bar t = i/\delta\)。两条级数和(几何、对数)值得单独背。

未知利率:换元解多项式

教材原话:未知利率"among the most difficult to solve"。高次多项式没有求根公式——但考题给你的都能换元降成二次,剩下的交给 BA II Plus(CF/IRR,后续章节)。

换元解 i 考频 ★★

课本 p.80 · Ex 2.11 / 2.12

令 \(x=1+i\) 或 \(x=v^n\),价值方程化成 x 的二次方程,取合理根、舍平凡根。

Ex 2.11(x = 1+i) $$(1+i)^2+2(1+i)=4\ \Rightarrow\ x^2+2x-4=0\ \Rightarrow\ x=\sqrt5-1=1.236068\ \Rightarrow\ i=23.61\%$$
Ex 2.12(x = v⁵,因式分解 + 舍根) $$100v^5+200v^{15}=300v^{10}\Rightarrow 2v^{10}-3v^5+1=0\Rightarrow(2x-1)(x-1)=0\Rightarrow v^5=\tfrac12\Rightarrow i=14.87\%$$
⚠️舍根意识:Ex 2.12 另一根 \(v^5=1\Rightarrow i=0\),题设 i 为正利率,舍。凡换元必检查每个根是否合题意。

换个角度看懂它

💡 直觉例子

比较日 = 数钱的桌子。桌子随便挑——今天这张、两年后那张、甚至 122 年后的那张——只要所有钱都搬到同一张桌子上,数出的结论一模一样。价值方程就是"这张桌子上,进的钱 = 出的钱"。

等时法 = 钱群的重心。把每块钱看作站在时间轴上的人,\(\bar t\) 是按体重(金额)加权的重心。折现曲线是凸的,所以重心总落在精确答案右边一点(偏晚)——这就是"恒偏大"的几何原因。

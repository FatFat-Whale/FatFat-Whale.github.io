年金家族辨析:一条时间线定乾坤 L03–L04

$a$、$\ddot a$、$s$、$\ddot s$、${}_{m|}a$、$1/i$……符号越背越乱?年金符号混淆的病根只有一个:没把付款点画在时间线上。期末/期初/递延/永续,只是同一串付款在时间线上的四种摆法——付款点画出来,该用哪个符号自己会跳出来。本页先看四联图,再两两对比换算关系,然后用「攒退休金 + 还房贷」五道题走完年金一生,最后给读题决策树。

💡 概念标签可点弹卡;所有数字手算可验;做年金题的第一动作永远是画时间线、标付款点,符号只是图的速记。

配套章节: L03 年金 · L04 复杂年金

第一幕 · 全景

四种年金 = 同一串付款的四种摆法

全景符号家族总表:站在哪、朝哪看

全家只有一张底牌:

$$a_{\overline{n}|}=\frac{1-v^n}{i}=\underbrace{\frac1i}_{\text{永续}}-\underbrace{v^n\cdot\frac1i}_{\text{推迟 }n\text{ 年的永续}}$$

白话:n 期年金 = 一条永续掐掉 n 年后的尾巴。其余成员全是这张底牌的平移(乘 $(1+i)$ 或 $v^m$)或换端读数(乘 $(1+i)^n$)。

符号站在哪个时点朝哪看闭式白话
$a_{\overline{n}|}$$t=0$朝前看 n 笔期末付款$\dfrac{1-v^n}{i}$n 笔 1 元折回今天
$\ddot a_{\overline{n}|}$$t=0$首笔就踩在脚下$\dfrac{1-v^n}{d}$每笔早一年 → 各多值一年利息
$s_{\overline{n}|}$$t=n$(末笔时点)朝回看已存的 n 笔$\dfrac{(1+i)^n-1}{i}$存完那刻的账户余额
$\ddot s_{\overline{n}|}$$t=n$(末笔后一年)朝回看$\dfrac{(1+i)^n-1}{d}$每笔都多滚了一年
$a_{\overline{\infty}|}$$t=0$朝前看无穷$1/i$只花利息,本金永不动
$\ddot a_{\overline{\infty}|}$$t=0$当场先领一笔$1/d$$=1+1/i$:先领 1,剩下是普通永续
读符号口诀:字母定端点($a$ 站 0 看现值,$s$ 站 n 看积累),两点定期初($\ddot{}$ = 首笔提前),前标定递延(${}_{m|}$ = 整串右移 m)。
第二幕 · 分组对比

两两换算:全家只差几个 $(1+i)$

对比①immediate vs due:每笔各早收一年利息
$$\ddot a_{\overline{n}|}=a_{\overline{n}|}\,(1+i),\qquad \ddot s_{\overline{n}|}=s_{\overline{n}|}\,(1+i)$$

白话:期初 = 整串付款左移一格 → 每笔各早付一年、各多滚一年利息 → 总值恰好整体乘一个 $(1+i)$。等价路径:分母 $i$ 换 $d$(因 $d=iv$,除以 $v$ 与乘 $(1+i)$ 是同一件事)。

错位恒等式时间线读法
$\ddot a_{\overline{n}|}=1+a_{\overline{n-1}|}$期初年金 = 当场领 1 元 + 剩下 n−1 笔期末年金
$\ddot s_{\overline{n}|}=s_{\overline{n+1}|}-1$多滚一年的积累 = 多存一期再退掉最后那笔本金
考场策略:只背 immediate 一套,见到 "beginning of each year" 最后乘 $(1+i)$——比单独背 due 公式少一半记忆量、少一处出错点。
对比②PV vs AV:同一串钱在两端的读数
$$s_{\overline{n}|}=a_{\overline{n}|}\,(1+i)^n,\qquad \ddot s_{\overline{n}|}=\ddot a_{\overline{n}|}\,(1+i)^n$$

价值方程视角:比较日随便挑,挑好后所有现金流用 $(1+i)^t$ 搬到同一天再对账——$a$ 和 $s$ 只是「比较日选 0 还是选 n」的两个特例。

读题抓问法:「现在值多少 / how much to invest today」→ PV 用 $a$;「到期攒了多少 / accumulated value」→ AV 用 $s$。别算完 $s$ 又手滑乘 $v^n$。
对比③递增递减家族:付款不再水平
题眼付款串PV 公式手感
"increases by 1 each year"$1,2,\dots,n$$(Ia)_{\overline{n}|}=\dfrac{\ddot a_{\overline{n}|}-nv^n}{i}$口诀「ä 在上,减 $nv^n$,除 $i$」
"decreases by 1 each year"$n,n-1,\dots,1$$(Da)_{\overline{n}|}=\dfrac{n-a_{\overline{n}|}}{i}$口诀「n 在上,减 $a$,除 $i$」
配对自检两串对应相加,每期都是 $n{+}1$$(Ia)+(Da)=(n{+}1)\,a_{\overline{n}|}$考场验算神器
"grows k% per year"$1,\,(1{+}k),\,(1{+}k)^2,\dots$(首笔 1 在期末)$\dfrac{1-\left(\frac{1+k}{1+i}\right)^{\!n}}{i-k}$增长抵掉一截利率:净折现率约 $i-k$
时间线上看:等差是「每年再叠一层水平年金」,等比是「每笔按 $(1+k)$ 长个子」——画两三笔付款标上金额,选公式就不会串。等比公式 $i=k$ 时分母为 0,退化为 $PV=n\,v$(每笔现值相等)。
对比④永续特权:$n\to\infty$ 后公式反而最短
$$a_{\overline{\infty}|}=\frac1i,\qquad \ddot a_{\overline{\infty}|}=\frac1d=1+\frac1i,\qquad (Ia)_{\overline{\infty}|}=\frac1i+\frac1{i^2},\qquad \text{等比永续}=\frac1{i-k}\ (k<i)$$

白话三连:

  1. $1/i$:本金 $1/i$ 每年恰好生息 1 元——只花利息、本金永不动,所以能永续。
  2. $1/d$:期初版 = 当场先领 1 元,剩下就是普通永续 → $1+1/i$。
  3. $1/i+1/i^2$:付款 $1,2,3,\dots$ = 每年新开一条「每年 1 元」的永续;第一条值 $1/i$,而「每年开一条 $1/i$」本身又排成一条永续 → 再加 $\frac{1/i}{i}$。
为什么最短:$n\to\infty$ 时 $v^n\to0$,底牌 $\frac{1-v^n}{i}$ 直接掉成 $\frac1i$——尾巴没了,公式只剩头。
第三幕 · 贯穿场景

攒退休金 + 还房贷:五问走完年金一生

Q1期末攒钱:$s_{\overline{30}|}$ 完整算一遍

30 岁的你决定每年年末存 1 万元进退休账户,连存 30 年,年有效利率 $i=5\%$。

第 30 笔存进去那一刻,账户里一共多少钱?

  1. 画时间线选符号:付款点在 $t=1..30$(期末),问「存完那刻的余额」→ 站在 $t=30$ 朝回看,用 AV:$s_{\overline{30}|}$。
  2. 套底牌:$s_{\overline{30}|}=\dfrac{(1+i)^{30}-1}{i}=\dfrac{1.05^{30}-1}{0.05}=\dfrac{4.32194-1}{0.05}=66.4388$。
  3. 到手:$1\text{万}\times 66.4388=\mathbf{66.44\ 万}$——本金只投了 30 万,利息滚出 36.44 万,复利占了大半壁江山。

N=30, I/Y=5, PV=0, PMT=−10000, CPT FV → 664,388.48

公式白话:分子 $(1+i)^n-1$ 是「1 元躺 n 年长出的总利息」,除以 $i$ 摊回「每年存 1 元」的口径——$s$ 就是这么来的。
Q2期初对比:同样的钱,早存一年值多少

同一计划改成每年年初存 1 万(其余不变),仍在原来那个时点(第 30 年年末)看余额。

比 Q1 多攒多少?

  1. 时间线上看:整串付款左移一格 → 每笔各多滚一年 → 整体乘 $(1+i)$:$\ddot s_{\overline{30}|}=s_{\overline{30}|}\times1.05=66.4388\times1.05=69.7608$。
  2. 到手:$69.76$ 万,比 Q1 多 $66.4388\times0.05=3.3219\to\mathbf{3.32\ 万}$——正好等于「Q1 的整锅余额再吃一年利息」。

2ND [BGN] 2ND [SET] 切 BGN 模式,其余同 Q1,CPT FV → 697,607.90(算完记得切回 END!)

高频坑:due 题按 immediate 算完忘乘 $(1+i)$——选项里必埋着那个偏小的错误答案等你跳;计算器 BGN 模式用完不切回,下一题全错。
Q3房贷月供:名义利率先换月有效

贷款 100 万买房,期限 30 年、按月等额还款,名义年利率 $6\%$ 按月计息($i^{(12)}=6\%$)。

月供多少?

  1. 先对齐节拍:付款按月,利率必须换成月有效:$j=\dfrac{6\%}{12}=0.5\%$,期数 $n=30\times12=360$ 个月。
  2. 贷款 = 月供的 PV(站在放款日):$P\cdot a_{\overline{360}|0.5\%}=100\text{万}$。
  3. $a_{\overline{360}|}=\dfrac{1-1.005^{-360}}{0.005}=\dfrac{1-0.16604}{0.005}=166.79$。
  4. $P=\dfrac{1{,}000{,}000}{166.79}=\mathbf{5{,}995.51}$ 元/月;30 年总还款 $5{,}995.51\times360\approx215.8$ 万——利息比本金还多。

N=360, I/Y=0.5, PV=1000000, FV=0, CPT PMT → −5,995.51

头号坑:I/Y 填的是每期利率——期是月就填 0.5,不是 6。凡是「付款频率 ≠ 利率标注频率」的题,第一步永远是换到付款期的有效利率,再当普通年金做。
Q4递延年金:攒到 65 岁再开领,两步折现

今天你 30 岁,想一次性存入一笔钱,从 65 岁生日起每年领 10 万、连领 20 年(65 岁生日当天领第一笔),$i=5\%$。

今天要存多少?

  1. 画时间线:今天 $t=0$,首笔在 $t=35$,共 20 笔——站在 $t=35$ 看,领取段是一条期初年金。
  2. 第一步(年金压缩):$\ddot a_{\overline{20}|}=a_{\overline{20}|}\times1.05=12.4622\times1.05=13.0853$ → 65 岁时点需要 $10\times13.0853=130.85$ 万。
  3. 第二步(整体折现):$130.8532\times v^{35}=130.8532\times1.05^{-35}=130.8532\times0.18129=\mathbf{23.72\ 万}$。
  4. 换条路验算(视作递延 34 年的期末年金,首笔仍在 $t=35$):${}_{34|}a_{\overline{20}|}=v^{34}a_{\overline{20}|}=0.19035\times12.4622=2.3722$ → 23.72 万 ✓ 两种摆法读数相同。

两步:BGN 模式 N=20, I/Y=5, PMT=100000, CPT PV → 1,308,532;再 N=35, PMT=0, FV=1308532, CPT PV → 237,224

递延套路:先在首笔附近把年金压成一坨,再整体 $v^m$ 折回今天——两步各自简单,漏第二步(只算到 65 岁时点就交卷)是经典丢分。
Q5永续奖学金:一步除法,递增换分母

校友想捐一笔本金设奖学金:每年年末发 5 万,永远发下去,基金年收益 $i=4\%$。

本金要多少?若奖学金每年递增 $2\%$(跟上通胀)呢?

  1. 水平永续:$PV=\dfrac{C}{i}=\dfrac{5}{0.04}=\mathbf{125\ 万}$——白话:125 万每年生息恰好 5 万,只花利息、本金永不动,自然发得到天荒地老。
  2. 期初版顺手看:改每年初发则 $PV=C/d=C(1+i)/i=125\times1.04=130$ 万——多出的 5 万就是当场先发的那笔。
  3. 递增 2%:$PV=\dfrac{C}{i-k}=\dfrac{5}{0.04-0.02}=\mathbf{250\ 万}$——每年利息里要留 $2\%$ 给本金「长个子」以支撑明年更高的奖学金,可持续花的只剩 $i-k$。
  4. 永续题不用计算器:$N=\infty$ 没法输,本来也就一步除法。
发散警报:递增永续要求 $k<i$;算出负分母说明读错题(或者那题根本是有限期等比,回去看年数)。
收官 · 速查

读题 30 秒:年金题拆解口诀

题面信号符号与动作
"at the end of each year"期末:$a_{\overline{n}|}$ / $s_{\overline{n}|}$,首笔在 $t=1$
"at the beginning of each year"期初:按 immediate 算完 ×$(1+i)$(或分母 $i\to d$)
"first payment at time m+1 / deferred m years"递延:${}_{m|}a_{\overline{n}|}=v^m a_{\overline{n}|}$,两步折现
"payments continue forever / perpetuity"$1/i$(期末)或 $1/d$(期初);递增 1 元:$1/i+1/i^2$
"accumulated value / balance after last deposit"AV:$s=a\,(1+i)^n$,站在 $t=n$ 读数
"increases by 1 / by k% each year"$(Ia)$ 等差 / 等比公式,分母 $i$ 或 $i-k$
"monthly payments, nominal annual rate"先换付款期有效利率($i^{(12)}/12$),n 用月数
高频坑疫苗
月供题拿年利率直接算付款按月 → 利率必换月有效,$n$ 必换月数(Q3)
due 忘乘 $(1+i)$ / BGN 模式忘切回答案总偏小一点点;算完 due 立刻切回 END(Q2)
递延两步折现漏一步压缩到首笔附近后,还欠一趟 $v^m$ 回今天(Q4)
首笔时点认错"deferred m 年 + 期末" → 首笔在 $m+1$,不在 $m$
递增永续 $k\ge i$ 硬套公式分母出负数 = 读错题,回去重画时间线(Q5)

自测:合上页面,把 $i$ 改成 4%、Q4 改成 60 岁开领,Q1–Q5 重算一遍;每题第一笔永远先画时间线、标付款点,再选符号。

错拍年金

推疏:\(j=(1+i)^k-1\) 或 \(a_{\overline{n}|}/s_{\overline{k}|}\)

密:\(a^{(m)}=\frac{1-v^n}{i^{(m)}}=\frac{i}{i^{(m)}}a\);系数=期内总付款

符号即公式

PV 分子 1−vⁿ · AV 分子 (1+i)ⁿ−1

分母:a→i · ä→d · (m)→i^(m)/d^(m) · bar→δ

等差(P&Q)

\(PV=Pa_{\overline{n}|}+Q\frac{a_{\overline{n}|}-nv^n}{i}\)

\((Ia)=\frac{\ddot a-nv^n}{i}\) · \((Da)=\frac{n-a}{i}\) · \((Is)=\frac{s_{\overline{n+1}|}-(n+1)}{i}\)

TVM:PMT=P+Q/i,FV=−nQ/i

增额永续

\((Ia)_\infty=\frac1i+\frac1{i^2}\) · P&Q 版 \(\frac Pi+\frac Q{i^2}\)

先增后平:\(\ddot a_{\overline{n}|}/i\)

等比 / 连续

推\(PV=\frac{1-(\frac{1+k}{1+i})^n}{i-k}\);\(i'=\frac{i-k}{1+k}\)

\(\bar a=\frac{1-v^n}{\delta}\) · \((\bar I\bar a)=\frac{\bar a-nv^n}{\delta}\)

回文 / 0%

\(1..n..1:\ a_{\overline{n}|}\ddot a_{\overline{n}|}\);time 0 起 \((\ddot a)^2\)

0%:v=1、a=s=n、(Ia)=n(n+1)/2,和对不上即排除

🎯 本章核心考点

  • 付款疏于/密于计息期都能双法处理;m-thly 符号系数一眼验对(÷m=单笔,系数 = 一个计息期总付款)。
  • 等差年金 P&Q 通式 + TVM 直按;\((Ia)\)/\((Da)\)/增额永续闭式随手写。
  • 等比年金换调整利率 \(i'\) 一键级别化;连续/变力年金列积分。
  • 回文年金乘积捷径 + 0% 检验秒排符号选项。

错拍年金:两法 + m-thly 陷阱

付款每 5 年一笔(疏)或每月一笔(密),计息却按年——把钟表调到同拍,或改造付款本身。

法一:换等效利率(数值题首选) 考频 ★★★

课本 p.151–153
★ 必背Equivalent Rate $$j=(1+i)^k-1\ (\text{疏})\qquad j=(1+i)^{1/m}-1\ (\text{密})\ \Rightarrow\ \text{当普通年金 TVM 直按}$$

Ex 4.1:每 5 年末付 1 共 40 年(5%):\(j=1.05^5-1=27.6282\%\),\(PV=a_{\overline{8}|j}=3.1054\)。名义利率先化期利率再合成(Ex 4.2 的 4% 季复利坑)。

法二:Fission / Fusion(符号题必会) 考频 ★★★

课本 p.153–158

疏 → 裂变:每 k 期一笔 1 拆成每期 \(1/s_{\overline{k}|}\);密 → 聚变:一期内 m 笔 1/m 聚成期末 \(i/i^{(m)}\)。

★ 必背Fission / m-thly 符号 $$PV_{\text{每 }k\text{ 期末付 }1}=\frac{a_{\overline{n}|}}{s_{\overline{k}|}}\qquad a^{(m)}_{\overline{n}|}=\frac{1-v^n}{i^{(m)}}=\frac{i}{i^{(m)}}a_{\overline{n}|}\qquad \ddot a^{(m)}_{\overline{n}|}=\frac{1-v^n}{d^{(m)}}\qquad a^{(m)}_{\overline{\infty}|}=\frac{1}{i^{(m)}}$$
🗝️大笔拆小笔,小笔聚大笔,现值不动 — 拆:小笔滚 k 期恰凑回原笔;聚:早付的钱多吃利息,所以期末等价笔略大(i/i^(m) 大于 1)。
⚠️m-thly 陷阱(4c,p.158):\(a^{(m)}\) 前系数 = 一个计息期内付款总和。月付 $100 十年(5% 有效)是 \(1{,}200a^{(12)}_{\overline{10}|}\) 不是 \(100a^{(12)}\)。Ex 4.14 更毒:季付 $18,000 + 名义 6% 半年复利 ⇒ 计息期=半年、期内 2 笔 ⇒ \(36{,}000\,a^{(2)}_{\overline{16}|}\) at 3%——m 由"计息期内几笔"定,不由"每年几笔"定。

★ 必背Continuous Annuity + 分母表 $$\bar a_{\overline{n}|}=\frac{1-v^n}{\delta}=\int_0^n v^t\,dt=\frac{i}{\delta}a_{\overline{n}|}\qquad \text{分母: } a\to i,\ \ddot a\to d,\ {}^{(m)}\to i^{(m)}/d^{(m)},\ \bar{\ }\to\delta$$
🗝️看见双点想 d,看见 bar 想 δ — PV 分子永远 1−vⁿ,AV 分子永远 (1+i)ⁿ−1,符号本身就是公式(4f Memory Alert)。

双点约掉(4e):同一利率下 \(\frac{\ddot a_{\overline{n}|}}{\ddot a_{\overline{p}|}}=\frac{a^{(m)}_{\overline{n}|}}{a^{(m)}_{\overline{p}|}}=\frac{1-v^n}{1-v^p}\)——L03 的 2n 技巧对 m-thly 照用(Ex 4.16:\(a^{(12)}\) 版 vⁿ=0.5 → i^(12)=5% → i=5.1162%)。利率不同则全失效。

⚠️变利率 s̄ 陷阱(4g,呼应 1g):非复利世界 t 时存 1 到 n 累积 \(\frac{a(n)}{a(t)}\),不是 a(n−t)。Ex 4.18 诱答 6.20,正解 7.34。

等差年金:P&Q 一族

付款 P, P+Q, P+2Q, …。一条通式生一族;计算器有专用变形,考场直按。

P&Q 通式与增减额闭式 考频 ★★★

课本 p.185–190
★ 必背P&Q + 三件套 $$PV=P\,a_{\overline{n}|}+Q\,\frac{a_{\overline{n}|}-nv^n}{i}\qquad (Ia)_{\overline{n}|}=\frac{\ddot a_{\overline{n}|}-nv^n}{i}\qquad (Da)_{\overline{n}|}=\frac{n-a_{\overline{n}|}}{i}\qquad (Is)_{\overline{n}|}=\frac{s_{\overline{n+1}|}-(n+1)}{i}$$
★ 必背增额永续 + 先增后平 $$(Ia)_{\overline{\infty}|}=\frac{1}{id}=\frac1i+\frac1{i^2}\qquad PV_{P\&Q\ \text{永续}}=\frac{P}{i}+\frac{Q}{i^2}\qquad PV_{1,2,\dots,n,n,n,\dots}=\frac{\ddot a_{\overline{n}|}}{i}$$
🗝️增额年金 = n 个错峰起跑的级别年金叠罗汉 — 所以分子藏着 ä;增额永续 = 平铺永续(1/i)+ "永续的永续"(1/i²)。
🖩Calculator Notes #6(p.191–192)TVM 直按:变形 \(PV=(P+\frac{Q}{i})a_{\overline{n}|}-\frac{nQ}{i}v^n\) ⇒ PMT = P+Q/i,FV = −nQ/i。例:1..15 @7.5%:15 N 7.5 I/Y 14.333 PMT -200 FV CPT PV → 58.93。(Is):CPT FV 后 −(n+1) 再 ÷i。
例 4.23课本 p.189 · 增额永续反解 i

首付 3、年增 2 的永续年金 PV=406.81。求 i。

\(\frac3i+\frac2{i^2}=406.81\Rightarrow 406.81i^2-3i-2=0\)。

i = 7.39%

等比年金:调整利率一键级别化

付款逐年 ×(1+k)(典型:通胀挂钩 COLA)。付款在涨、折现在压,对冲后剩净利率 i′——世界重新变平。

通式 + Trick 考频 ★★★

课本 p.216–217
★ 必背Geometric Annuity $$PV=\frac{1-\bigl(\frac{1+k}{1+i}\bigr)^n}{i-k}\qquad i'=\frac{i-k}{1+k}\ \Rightarrow\ PV=(\text{首付})\,v\,\ddot a_{\overline{n}|i'}$$

Ex 4.26/4.27 双解对照:15 年、首付 1,000、×1.03、i=5.25% → 12,304.56(通式与调整率法同值)。递减几何(k 负,Ex 4.28:每年降到 90%)直接套。与 L05 通胀年金同源:k=通胀率时 i′ 就是实际利率。

金水机、回文年金与 0% 检验

连续变额年金(金水机) 考频 ★★

课本 p.227–230
★ 必背微元法 $$(\bar I\bar a)_{\overline{n}|}=\int_0^n t\,v^t dt=\frac{\bar a_{\overline{n}|}-nv^n}{\delta}\qquad PV=\int_0^n f(t)\,e^{-\int_0^t\delta_r dr}dt\qquad AV=\int_0^n f(t)\,e^{\int_t^n\delta_r dr}dt$$
🗝️水龙头刻度 = 年流速 — PV 是每滴水按落地时间打折的总量;AV 的指数从付款时刻积到期末(\(\int_t^n\)),别写成 \(\int_0^n\)。

回文年金 + 0% 检验 考频 ★★

课本 p.235–241
★ 必背Palindromic + 0% Test $$PV_{1..n..1}=a_{\overline{n}|}\cdot\ddot a_{\overline{n}|}\qquad \text{time 0 起跑}=(\ddot a_{\overline{n}|})^2\qquad \text{0\%: } v=1,\ a=s=n,\ (Ia)=\tfrac{n(n+1)}{2}$$

回文直觉:金字塔切成一摞错峰起跑的同款级别年金 ⇒ 乘积。0% 检验(4m,考场锦囊):符号选择题把 i=0 代入——一切退化成付款算术和,选项对不上直接排除(Ex 4.36 五选项秒杀三个)。必要非充分;特定利率或分母含 i/d/δ 时失效。

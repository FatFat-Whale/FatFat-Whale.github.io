amortize 的词根是 mort(死亡)——逐期付款把贷款"杀死"。每期付款先付利息、剩余还本;本金偿还额按 (1+i) 等比放大。摊还表这台机器学会了,Ch7 债券直接换皮沿用。教材开篇 Concept Alert:此章是必须运用自如的基本功。对照课本 Chapter 6 精读。

Ch6 习题最多章(110 题) 课本 p.271–330(理论 ≈ 20 页) 五小节:摊还 / 变付 / 等额本金 / 尾款 / 再融资

★ 本章拿走 4 句话

  1. 递推三件套:\(I_t=iB_{t-1}\)、\(P_t=R_t-I_t\)、\(B_t=B_{t-1}-P_t\);本金偿还额等比,公比 (1+i)。
  2. 余额两视角:前瞻 = 剩余付款 PV(\(Ra_{\overline{n-t}|}\));追溯 = 原贷款滚息 − 已付积累(\(L(1+i)^t-Rs_{\overline{t}|}\))。
  3. 期数非整数收尾:balloon(第 ⌊N⌋ 期多付)或 drop(第 ⌈N⌉ 期少付),两者差一年利息 \(X_2=(1+i)X_1\)。
  4. 再融资通法:旧条款算时点余额,令其等于新条款付款的 PV。

§6a · 课本 p.271–293

摊还:利息优先,剩余还本

为什么要拆利息/本金?按揭利息可抵税;贷方(精算师)账上只有利息部分是真赚的。

递推三件套与摊还表 考频 ★★★

课本 p.271–274

每期付款 = 利息(按期初余额)+ 本金(剩余部分);余额随本金递减。

★ 必背递推三件套 $$I_t=i\,B_{t-1}\qquad P_t=R_t-I_t\qquad B_t=B_{t-1}-P_t$$
🗝️本金等比,公比 1+i — \(P_{t+s}=P_t(1+i)^s\);贷款额 = 全部本金之和。步数别多一:P₁ 到 P₁₀ 走 9 步(×1.05⁹)。
★ 必背任意一行直取(等额付款 R = L/a) $$I_t=R\,(1-v^{\,n-t+1})\qquad P_t=R\,v^{\,n-t+1}\qquad B_t=R\,a_{\overline{n-t}|}$$

总利息 = 总付款 − 原贷款额。摊还表与贷款额成比例——单位表算好,乘 L/a 缩放。

例 6.3课本 p.275 · 等比双向推

15 年贷款、7%,第 5 期本金 $187。求贷款额。

从 P₅ 向前折现 4 步、向后累积 10 步:\(L=187(a_{\overline{4}|}+s_{\overline{11}|})=187(3.3872+15.7836)\)。

= $3,585

步骤用的公式 / 定理出处
① 认题型:知单期本金反推贷款额 → 贷款额 = 全部本金之和\(L=\sum_{t=1}^{15}P_t\)本章

课本 p.271 / p.276
★ 必背Prospective / Retrospective $$B_t^{p}=R\,a_{\overline{n-t}|}\qquad B_t^{r}=L(1+i)^t-R\,s_{\overline{t}|}$$
⚠️追溯法漏滚息(Ex 6.4 点名):\(1000-125s_{\overline{8}|}\) 是错的——原贷款必须先滚 \((1.08)^8\) 再减。付款不足以知期限时只能追溯;付款串再乱、只要未来规整就前瞻(Ex 6.5:60 年怪付款串,第 50 期后余额 = \(1200a_{\overline{10}|}\),前面全不用看)。

§6b–6c · 课本 p.294–310

变动付款与等额本金

摊还表是台傻瓜机器:任何现值 = 贷款额的付款流,它都按"利息优先"运转,最后一格自动归零。

负摊还与付款增量式 考频 ★★

课本 p.294–295

付款 < 当期利息 ⇒ 本金列为负、余额上升(negative amortization),欠付利息被"资本化"滚入本金。

相邻期本金关系(付款不水平时) $$P_t=(1+i)P_{t-1}+(R_t-R_{t-1})$$

Ex 6.9:第 t 期付 2,000、本金 1,000;下期付 3,000 ⇒ \(P_{t+1}=1{,}000(1.1)+1{,}000=2{,}100\)——多付的钱全部进本金。付款水平时退化为等比。

等额本金 ≠ 等额付款 考频 ★★

课本 p.304–305

每期还固定本金 X + 按余额的利息 ⇒ 余额线性下降、总付款逐期递减(等差)。

第 t 期付款与定价 $$R_t=X+i[L-(t-1)X]\qquad \text{定价(收益率 j)}=Xa_{\overline{n}|j}+iX(Da)_{\overline{n}|j}$$
🗝️每年还一块砖 + 为剩下的砖交租 — 砖变少租金变少;与等额付款(月供恒定、前期几乎全是利息)对照。总利息用等差和:\(i\,X(n+(n-1)+\cdots+1)\) 型。

Ex 6.12 对比:$100,000、10 年、5%——等额付款总利息 29,504.60,等额本金 27,500(余额降得快利息少),差 $2,004.60。

尾款:balloon 与 drop

CPT N 常出非整数(如 14.2067)——最后一片香肠切不齐:要么最后一刀切厚点,要么多切一刀薄的。

Balloon / Drop 考频 ★★★

课本 p.311–313

balloon:第 ⌊N⌋ 期付款加大;drop:第 ⌈N⌉ 期付一笔小于正常付款的尾款。

★ 必背价值方程与两者关系 $$L=R\,a_{\overline{\lfloor N\rfloor}|}+X_1v^{\lfloor N\rfloor}\ (\text{balloon 追加})\qquad L=R\,a_{\overline{\lfloor N\rfloor}|}+X_2v^{\lceil N\rceil}\ (\text{drop})\qquad X_2=(1+i)X_1$$
🖩FV 符号读法(p.311,本节计算器核心):解出 N=14.2067 后,14 N CPT FV → −200.68(负 = 还要额外流出,balloon 总付 1,200.68);15 N CPT FV → +789.28(正 = 超付退回,drop = 1,000−789.28 = 210.72)。
展开:Example 6.13–6.15(取款版同理)

Colin 年存 1 共 10 次(5%),之后年取 1:FV=12.5779 转 PV,CPT N=20.3 ⇒ 20 次足额;第 21 年 drop 提取 = 1−.677714 = .322286;若第 20 年一次取完(balloon)= 1+.322286/1.05 = 1.306939(p.312)。

直觉 · 拓展

💡 直觉例子

每期月供 = 房租 + 赎楼款。先付"房租"(利息 iB),剩下才是真赎回;余额越还越小、房租越交越少,同样的月供里赎楼份额像雪球一样按 (1+i) 等比放大——这就是"前期还的几乎全是利息"的数学版。

负摊还 = 信用卡最低还款还不够利息:差额记账、雪球反向变大。amortize 词根 mort(死亡)——正常摊还是把贷款慢慢"杀死",负摊还是它在回血。

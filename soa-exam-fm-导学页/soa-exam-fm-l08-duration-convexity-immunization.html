第8章 · Duration, Convexity, and Immunization 久期、凸性与免疫

利率一动,固收资产的现值就动——动多少?久期量一阶(斜率),凸性量二阶(弯度),免疫用这两把尺给资产负债配平。教材原话:近年考题在本章大量出题,考前必须彻底复习。三层主线:度量 → 近似 → 防护。对照课本 Chapter 8 精读。

Ch8 新大纲高频章 课本 p.375–414(理论 ≈ 22 页) 习题 45 题(24+21,ASM 原创)

★ 本章拿走 4 句话

  1. Dmac = 现金流的 PV 重心时刻;零息债久期 = 期限;平价附息债 = \(\ddot a_{\overline{n}|}\)。
  2. \(D_{mod}=vD_{mac}\);两个一阶近似里 Macaulay 版更准:\(P(i)\approx P(i_0)\bigl(\frac{1+i_0}{1+i}\bigr)^{D_{mac}}\)。
  3. 考试默认:duration = Macaulay,convexity = modified——两个默认方向相反(p.397 术语表)。
  4. Redington 三条件:PV 等、久期等、资产凸性更大(资产现金流跨骑负债两侧);full immunization = 前后各一笔,防任意幅度。

核心

核心知识点及考题策略

先扫一遍公式卡热身(带「推」标的悬停看推导);学完回来对照「考点 + 套路」自检,这一节全拿下才算过关。

久期

推\(D_{mac}=\frac{\sum t v^t A_t}{P}\) · \(D_{mod}=vD_{mac}\)

零息=n · 永续=\(\frac{1+i}{i}\) · 平价债=\(\ddot a_{\overline{n}|}\) · 增长股=\(\frac{1+i}{i-g}\)

一阶近似

推修正:\(P\approx P_0[1-\Delta i\,D_{mod}]\)(恒低估)

Macaulay:\(P\approx P_0(\frac{1+i_0}{1+i})^{D_{mac}}\)(更准)

组合与时间

组合久期 = 价格加权平均

随时间锯齿下降,付息瞬间跳升

凸性

\(C_{mod}=\frac{\sum t(t+1)v^{t+2}A_t}{P}\) · \(C_{mac}=\frac{\sum t^2v^tA_t}{P}\)

\(C_{mod}=\frac{C_{mac}+D_{mac}}{(1+i)^2}\)

Redington

\(P_A=P_L\) · \(P'_A=P'_L\) · \(P''_A\gt P''_L\)

= 盈余在 i₀ 相对极小;第三条件 ⇔ t² 加权比较

Full / Dedication

Full:PV 等+久期等+前后各一笔(防任意幅度)

Dedication:\(A_t=L_t\),最远期倒推;需 rebalance

🎯 本章核心考点

  • 任给现金流算 Dmac/Dmod/Cmac/Cmod;五种标准资产的久期闭式随手写。
  • 用两个一阶近似估新价格,并说出谁更准、误差方向(修正版恒低估)。
  • Redington 三条件列联立解资产面值;判断第三条件成败(跨骑 vs 夹心)。
  • full immunization 与 dedication(精确匹配,从最远期倒推)两套配平法。

§8a–8f · 课本 p.375–394

久期:PV 的重心与斜率

引子:6%→6.1%,10 年零息债跌 0.94%、20 年跌 1.87%——敏感度按百分比算(不是美元),且 ∝ 期限。单笔现金流的久期就是它的剩余年数。

Macaulay 久期 考频 ★★★

课本 p.377–381

把每笔现金流的 PV 当砝码摆上时间轴,天平支点 = Dmac;数学上等价于 PV 对利息力 δ 的敏感度。

★ 必背Macaulay Duration $$D_{mac}=\frac{\sum t\,v^t A_t}{\sum v^t A_t}=-\frac{dP/d\delta}{P}$$
五种标准资产(Ex 8.5,i=5%) $$\text{零息债}=n\quad \text{永续 immediate}=\frac{1+i}{i}\quad \text{平价附息债}=\ddot a_{\overline{n}|}\quad \text{增长股}=\frac{1+i}{i-g}\quad \text{按揭}=\frac{(Ia)_{\overline{n}|}}{a_{\overline{n}|}}$$
🗝️券息越高,久期越短 — 早期现金流权重大,重心前移(Ex 8.6:5% 券 8.11 年,10% 券 7.27 年,零息 10 年)。

组合久期 = 成分久期的价格加权平均(Ex 8.12)。时间推移:久期锯齿下降——渐降、每逢付息日跳升一齿(Ex 8.13:8.11→7.11→7.46)。SOA 规定:无修饰词的 "duration" = Macaulay。

Modified 久期与两个一阶近似 考频 ★★★

课本 p.381–384

对 i 求导得 modified:\(D_{mod}=vD_{mac}\)("v for variation")。近似新价格有两条路,Macaulay 版更准。

★ 必背两个一阶近似 $$\text{修正版: } P(i)\approx P(i_0)\bigl[1-(i-i_0)D_{mod}(i_0)\bigr]\qquad \text{Macaulay 版: } P(i)\approx P(i_0)\Bigl(\frac{1+i_0}{1+i}\Bigr)^{D_{mac}(i_0)}$$

Ex 8.10/8.11 对照:15 年 8% 平价债 → 7.92%,精确 1,006.88;修正近似 1,006.85(永远低估——切线在凸曲线下方);Macaulay 近似 1,006.87(误差约 1/3,方向天然自洽)。平价债专属捷径:\(D_{mod}=v\ddot a_{\overline{n}|}=a_{\overline{n}|}\)——一撇之差常考。

★ 必背Convexity $$C_{mod}=\frac{P''(i)}{P(i)}=\frac{\sum t(t+1)v^{t+2}A_t}{P(i)}\qquad C_{mac}=\frac{\sum t^2v^tA_t}{P(i)}\qquad C_{mod}=\frac{C_{mac}+D_{mac}}{(1+i)^2}$$

Redington 免疫(防小幅) 考频 ★★★

课本 p.397–401

三充分条件,在 i=i₀ 处成立、只保小幅变动:

★ 必背Redington 三条件 $$P_A=P_L\qquad P'_A=P'_L\ (D^A=D^L)\qquad P''_A\gt P''_L\ (C^A\gt C^L)$$
🗝️盈余在 i₀ 处取相对极小 — 等价写法 P=P_A−P_L:P=0、P′=0、P″>0;利率小幅上下波,盈余都上升。第三条件 ⇔ \(\sum t^2v^tA_t \gt \sum t^2v^tL_t\)(Ex 8.21)。
例 8.19课本 p.399–400 · 经典解法模板

负债:10 年期末年金每年 $10;资产:2 年与 9 年零息债(面值 X、Y),i=10%。求 X、Y。

条件一 \(v^2X+v^9Y=10a_{\overline{10}|}\);条件二 \(2v^3X+9v^{10}Y=10v(Ia)_{\overline{10}|}\)(\(\frac{d}{di}v^n=-nv^{n+1}\))。联立解出。

X = 45.40 · Y = 56.41

Full Immunization 与 Dedication 考频 ★★

课本 p.402–405

Full:PV 等 + 久期等 + 负债现金流前后各一笔资产 ⇒ 防任意幅度变化(多笔负债逐笔配三件套)。Dedication:干脆逐期精确匹配 \(A_t=L_t\),利率爱怎么动怎么动。

Ex 8.22(full):负债 2 年 $1,000,用 1 年与 4 年零息配平 → X=620.16、Y=385.21;实测 i→0 与 i→100% 两端都盈余。Ex 8.24(dedication):从最远期负债倒推逐只附息债面值,票息回填前期。免疫非一劳永逸:时间流逝+利率变动改变久期(8f 锯齿),需 rebalance(p.403)。

展开:dedication 为什么"说易行难"(p.403)

现金流入不总可预测(债券被 call、按揭提前还);流出常是估计(死亡/伤残/退保);市场未必有恰好期限的资产;严格匹配牺牲收益率。别名:cash flow matching / exact matching / absolute matching。

💡 直觉例子

久期是天平支点,凸性是翘板弯度。每笔现金流的 PV 是砝码,D_mac 是支点位置;券息越高、砝码越靠前,支点前移。凸性是曲线"上翘"的程度——切线(一阶)估价永远偏低,弯得越厉害差得越多。

免疫像搭跷跷板:PV 等 = 两边一样重;久期等 = 支点对齐;资产凸性更大 = 资产那边翘得更弯——利率小幅怎么晃,资产端都压得住。资产现金流要"跨骑"负债两侧才弯得起来,夹在中间就输。
